NOM

motd - Message du jour.  

DESCRIPTION

/etc/motd (Message Of The Day) est un fichier de texte ASCII dont le contenu est affiché par login(1) avant d'exécuter le shell.

"Motd" signifie "message of the day", et il contient généralement des messages destinés aux utilisateurs concernant l'administration du système (ajout de périphériques, arrêts prévus du système, nouvelles versions de logiciels, etc...) Il nécessite nettement moins d'espace disque qu'un courrier électronique envoyé à tous les utilisateurs.  

EXEMPLE

$ ls -l /etc/motd
-rw-r--r--   1 root     root        71 Oct 17 17:09 /etc/motd
$ cat /etc/motd
