NOM

fcloseall - Fermer tous les flux ouverts.  

SYNOPSIS

#define _GNU_SOURCE

#include <stdio.h>

int fcloseall(void);  

DESCRIPTION

La fonction fcloseall dissocie tous les flux ouverts des fichiers sous-jacents. Toutes les données en attente dans les buffers de sortie sont écrites d'abord en utilisant fflush(3). Notez que les flux standards (stdin, stdout et stderr) sont également fermés.  

VALEUR RENVOYÉE

Cette fonction renvoie toujours 0.  

VOIR AUSSI

close(2), fclose(3), fflush(3), fopen(3), setbuf(3)  

CONFORMITÉ

La fonction fcloseall est une extension GNU.
