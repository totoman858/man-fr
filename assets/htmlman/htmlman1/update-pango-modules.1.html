NOM

update-pango-modules - Script de mise à jour de la liste des modules Pango  

SYNOPSIS

update-pango-modules paquet version  

DESCRIPTION

update-pango-modules est un script pour la mise à jour de /etc/pango/pango.modules en utilisant pango-querymodules.
update-pango-modules reçoit un argument et /etc/pango/pango.modules sera généré pour cette version par pango-querymodules.  

VOIR AUSSI

pango-querymodules(1)
