NOM

tkpppoe - Interface graphique pour contrôler rp-pppoe.  

SYNOPSIS

tkpppoe

 

DESCRIPTION

tkpppoe est une interface graphique pour contrôler les liens PPPoE. Il travaille avec le paquetage RP-PPPoE et possède son propre manuel HTML.

 

FICHIERS

/etc/ppp/rp-pppoe-gui/connection-info
Contient les informations de connexion. Ce fichier n'est pas manuellement modifiable.

/etc/ppp/rp-pppoe-gui/passwd
Contient les mots de passe pour chacune des connexions. Ce fichier n'est pas manuellement modifiable.

/etc/ppp/rp-pppoe-gui/conf.*
Ces fichiers de configuration sont utilisés par pppoe-start. Ils sont de nouveau générés par tkpppoe chaque fois qu'un changement est effectué dans les propriétés de connexion.
