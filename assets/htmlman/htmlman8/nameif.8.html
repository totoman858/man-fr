NOM

nameif - Nommer les interfaces réseau à partir des adresses MAC.  

SYNOPSIS

nameif [-c fichier_config] [-s]
nameif [-c fichier_config] [-s] {interface adresse_mac}  

DESCRIPTION

nameif renomme les interfaces de réseau à partir des adresses MAC. Si aucun paramètre n'est fourni, il prend en compte le fichier /etc/mactab. Chacune de ses lignes contient un nom d'interface et une adresse MAC. Les commentaires sont autorisés s'ils commencent par le caractère #. Autrement, les interfaces définies en ligne de commande sont traitées. nameif considère l'interface avec l'adresse MAC indiquée et la renomme avec le nom donné.

Avec l'option -s, tous les messages d'erreur sont envoyés vers syslog. Avec l'option -c suivie d'un nom de fichier, celui-ci est lu à la place de /etc/mactab.  

NOTES

nameif doit être lancé avant d'activer l'interface, sinon il échouera.  

FICHIERS

/etc/mactab  

BOGUES

Il ne fonctionne actuellement que pour Ethernet.
