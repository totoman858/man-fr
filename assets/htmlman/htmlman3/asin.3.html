NOM

asin, asinf, asinl - Fonction Arc Sinus.  

SYNOPSIS

#include <math.h>

double asin (double x);
float asinf (float x);
long double asinl (long double x);
 

DESCRIPTION

La fonction asin() calcule l'Arc Sinus de x, c'est à dire la valeur dont le sinus est x. Si x se trouve en dehors de l'intervalle [-1, 1], asin() reverra NaN (Not a Number), et errno contiendra le code d'erreur EDOM.  

VALEUR RENVOYÉE

La fonction asin() renvoie l'Arc Sinus en radians, dans l'intervalle -PI/2 à PI/2 (bornes comprises).  

NOTES

Pour utiliser les fonctions de la bibliothèque mathématique, il ne faut pas oublier d'ajouter l'argument -lm sur la ligne de commande de cc(1) ou ld(1), lors de l'édition des liens.  

ERREURS

EDOM
x n'est pas dans l'intervalle [-1, 1].
 

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899. Les variantes float et long double sont des demandes C99.  

VOIR AUSSI

acos(3), atan(3), atan2(3), cos(3), sin(3), tan(3)
