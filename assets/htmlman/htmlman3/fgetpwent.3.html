NOM

fgetpwent - Lire un fichier de mots de passe.  

SYNOPSIS

#include <pwd.h>
#include <stdio.h>
#include <sys/types.h>

struct passwd *fgetpwent (FILE *stream);
 

DESCRIPTION

La fonction fgetpwent() renvoie une structure contenant les différents champs d'une ligne du fichier stream. Lors de la première invocation, elle renvoie le premier enregistrement, ensuite elle retourne successivement tous les autres enregistrements du fichier. Le fichier stream doit avoir le même format que /etc/passwd(5).

La structure passwd est définie dans <pwd.h> ainsi : 


struct passwd {
        char    *pw_name;     /* Nom d'utilisateur       */
        char    *pw_passwd;   /* Mot de passe            */
        uid_t    pw_uid;      /* ID utilisateur          */
        gid_t    pw_gid;      /* ID groupe               */
        char    *pw_gecos;    /* Vrai nom                */
        char    *pw_dir;      /* Répertoire de connexion */
        char    *pw_shell;    /* Shell à la connexion    */
};
 

VALEUR RENVOYÉE

La fonction fgetpwent() renvoie la structure de mot de passe, ou NULL si il n'y a plus d'enregistrement, ou si une erreur s'est produite.  

ERREURS

ENOMEM
Pas assez de mémoire pour allouer la structure.
 

FICHIERS

/etc/passwd
Base de données des mots de passe
 

CONFORMITÉ

SVID 3  

VOIR AUSSI
