NOM

undocumented - Liste des appels systèmes non documentés.  

DESCRIPTION

Le noyau 2.4.20 connait 253 appels-système pour l'architecture i386. La liste suivante indique ceux qui n'ont pas encore été documentés dans le manuel du programmeur Linux : exit_group, madvise1, rt_sigaction, rt_sigpending, rt_sigprocmask, rt_sigqueueinfo, rt_sigreturn, rt_sigsuspend, rt_sigtimedwait, ugetrlimit  

VOIR AUSSI

obsolete(2), syscalls(2), unimplemented(2)
