NOM

getloadavg - Renvoie les moyennes de la charge du système.  

SYNOPSIS

#include <stdlib.h>

int getloadavg(double loadavg[], int nelem);  

DESCRIPTION

La fonction getloadavg renvoie le nombre de processus présents dans la file des processus prêts du système. Ce nombre est une moyenne réalisée sur diverses périodes. Jusqu'à nelem acquisitions sont réalisées et affectées aux éléments successifs de loadavg[]. Le système impose un maximum de 3 échantillonages, représentant des moyennes sur respectivement la dernière minute, les 5 dernières minutes et les 15 dernières minutes.  

DIAGNOSTICS

Si la charge moyenne n'a pas pu être obtenue, on renvoie -1; sinon, le nombre d'acquisitions effectivement réalisées est renvoyé.  

VOIR AUSSI

uptime(1), proc(5)  

HISTORIQUE

La fonction getloadavg est apparue dans 4.3 Reno.
