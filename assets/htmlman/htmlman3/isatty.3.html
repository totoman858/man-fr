NOM

isatty - Vérifier si un descripteur se rapporte à un terminal.  

SYNOPSIS

#include <unistd.h>

int isatty (int desc);
 

DESCRIPTION

renvoie 1 si desc est un descripteur ouvert connecté à un terminal, ou 0 sinon.  

CONFORMITÉ

SVID, AT&T, X/OPEN, BSD 4.3  

VOIR AUSSI

fstat(2), ttyname(3)  

TRADUCTION

Christophe Blaess, 1996-2003.
