NOM

cosh, coshf, coshl - Fonction Cosinus Hyperbolique.  

SYNOPSIS

#include <math.h>

double cosh (double x);
float coshf (float x);
long double coshl (long double x);
 

DESCRIPTION

La fonction cosh() renvoie le Cosinus Hyperbolique de x, que l'on définit mathematiquement ainsi :

CosH (x) = (exp(x) + exp(-x)) / 2  

NOTES

Pour utiliser les fonctions de la bibliothèque mathématique, il ne faut pas oublier d'ajouter l'argument -lm sur la ligne de commande de cc(1) ou ld(1), lors de l'édition des liens.  

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899 (C99). Les variantes float et long double sont des demandes C99.  

VOIR AUSSI

acosh(3), asinh(3), atanh(3), sinh(3), tanh(3)
