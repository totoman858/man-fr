SINH

Section: Manuel du programmeur Linux (3)
Updated: 30 juillet 2003
Index Return to Main Contents
 

NOM

sinh, sinhf, sinhl - Fonction sinus hyperbolique.  

SYNOPSIS

#include <math.h>

double sinh (double x);

float sinh (float x);

long double sinh (long double x);
 

DESCRIPTION

La fonction sinh() renvoie le sinus hyperbolique de x, défini mathématiquement comme (exp(x) - exp(-x)) / 2.  

NOTES

Pour utiliser les fonctions de la bibliothèque mathématique, il ne faut pas oublier d'ajouter l'argument -lm sur la ligne de commande de cc(1) ou ld(1), lors de l'édition des liens.  

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899. Les variantes float et long double sont des demandes C99.  

VOIR AUSSI

acosh(3), asinh(3), atanh(3), cosh(3), tanh(3)
