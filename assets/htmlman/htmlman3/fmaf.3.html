NOM

fma - Multiplication et addition en virgule flottante.  

SYNOPSIS

#include <math.h>

double fma(double x, double y, double z);

float fmaf(float x, float y, float z);

long double fmal(long double x, long double y, long double z);
 

DESCRIPTION

La fonction fma() calcule x*y+z. Le résultat est arrondi en fonction du mode déterminé par la valeur de FLT_ROUNDS.  

CONFORMITÉ

C99  

VOIR AUSSI

drem(3)
