PWCONV

Section: Maintenance Commands (8)
Updated: 26 septembre 1997
Index Return to Main Contents
 

NOM

pwconv, pwunconv, grpconv, grpunconv - convertit de et vers les fichiers de mots de passe et groupes masqués.  

SYNOPSIS

pwconv
pwunconv
grpconv
grpunconv  

DESCRIPTION

Ces quatre programmes opèrent tous sur les fichiers mots de passe et groupes normaux et masqués : /etc/passwd, /etc/group, /etc/shadow, et /etc/gshadow.

pwconv crée shadow à partir de passwd et optionnellement d'un shadow existant.
pwunconv crée passwd à partir de passwd et shadow puis supprime shadow.
grpconv crée gshadow à partir de group et optionnellement d'un gshadow existant.
grpunconv crée group à partir de group et gshadow puis supprime gshadow.

Chaque programme acquiert les verrouillages nécessaires avant la conversion.

pwconv et grpconv sont similaires. D'abord, les entrées du fichier caché qui n'existent pas dans le fichier principal sont supprimées. Ensuite, les entrées cachées qui n'ont pas « x » comme mot de passe dans le fichier principal sont mises à jour. Toutes les entrées cachées manquantes sont ajoutées. Enfin, les mots de passe dans le fichier principal sont remplacés par « x ». Ces programmes peuvent être utilisés aussi bien pour une première conversion que pour une mise à jour du fichier caché si le fichier principal a été édité à la main.

pwconv utilisera les valeurs PASS_MIN_DAYS, PASS_MAX_DAYS, et PASS_WARN_AGE de /etc/login.defs lors de l'ajout de nouvelles entrées dans /etc/shadow.

De la même manière, pwunconv et grpunconv sont similaires. Les mots de passe du fichier principal sont mis à jour à partir du fichier caché. Les entrées qui existent dans le fichier principal mais pas dans le fichier caché sont laissées de coté. Le fichier caché est ensuite supprimé. Certaines informations vieillissantes sur les mots de passe sont perdues par pwunconv. Il convertira ce qu'il pourra.

 

BOGUES

Des erreurs dans les fichiers mots de passe ou groupes (comme les entrées invalides ou dupliquées) peuvent entrainer ces programmes dans des boucles infinies ou les faire échouer de manière étrange. Veuillez lancer pwck et grpck pour corriger de telles erreurs avant de convertir de ou vers les mots de passe ou groupes cachés.
