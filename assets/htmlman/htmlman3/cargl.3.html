NOM

carg, cargf, cargl - Fonctions arguments de nombres complexes.  

SYNOPSIS

#include <complex.h>

double carg(double complex z);

float cargf(float complex z);

long double cargl(long double complex z);

 

DESCRIPTION

Un nombre complexe peut être décrit par deux cordonnées réelles.

On peut utiliser les coordonnées rectangulaires et avoir z = x+iy, où x = creal(z) et y = cimag(z).

On peut aussi utiliser les coordonnées polaires et avoir z = r*cexp (i*a), où r = cabs(z) est l'argument (le rayon, la valeur absolue) de z, et a = carg(z) est le module (l'angle de phase) de z.

On a carg(z) = atan(creal(z)/cimag(z)).  

VALEUR RENVOYÉE

La valeur renvoyée est dans l'intervalle [-pi,pi].  

CONFORMITÉ

C99  

VOIR AUSSI

cabs(3), complex(5)
