NOM

getenv - Lire une variable d'environnement.  

SYNOPSIS

#include <stdlib.h>

char *getenv (const char *name);
 

DESCRIPTION

La fonction getenv() recherche dans la liste des variables d'environnement une chaîne correspondant à celle pointée par name. Les chaînes sont de la forme nom = valeur.  

VALEUR RENVOYÉE

La fonction getenv() renvoie un pointeur sur la valeur correspondante, dans l'environnement du processus, ou NULL s'il n'y a pas de correspondance.  

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899  

VOIR AUSSI
