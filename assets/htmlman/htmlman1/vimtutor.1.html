NOM

vimtutor - le tutoriel Vim  

SYNOPSIS


vimtutor [langue]  

DESCRIPTION

Vimtutor lance le tutoriel Vim. Le fichier tutor est d'abord recopié, afin qu'il puisse être édité sans altérer le fichier d'origine.

Le Tutoriel Vim sera utile aux personnes désirant faire leur premières armes avec Vim

L'argument optionnel [langue] est composé des deux premières lettres de la langue, par exemple « it » ou « es ». Si l'argument [langue] n'est pas précisé, la langue spécifiée par la locale en cours sera utilisée. Si un tutoriel écrit dans la langue demandée est disponible, il sera utilisé. Sinon, la version anglaise sera utilisée.

Vim est toujours lancé en mode compatible Vi.  

FICHIERS

/usr/local/lib/vim/tutor/tutor[.langue]
Le(s) fichier(s) texte du Tutoriel Vim
/usr/local/lib/vim/tutor/tutor.vim
Le script Vim chargé de la copie du fichier texte du Tutoriel Vim
