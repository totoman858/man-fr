NOM

locale.gen - Fichier de configuration pour locale-gen  

DESCRIPTION

Le fichier /etc/locale.gen liste les locales qui vont être générées par la commande locale-gen.

Chaque ligne est de la forme :

<locale> <charset>

où <locale> est une des locales données dans /usr/share/i18n/locales et <charset> est un des jeux de caractères listés dans /usr/share/i18n/charsets

La commande locale-gen génèrera toute les locales, en les plaçant dans /usr/lib/locale.
