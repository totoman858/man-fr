NOM

sigvec - Utiliser les signaux BSD.  

SYNOPSIS

#include <bsd/signal.h>

int sigvec(int sig, struct sigvec *vec, struct sigvec *ovec);  

DESCRIPTION

Cette interface est rendue obsolète par sigaction(2).

Sous Linux sivec est #defini avec sigaction, et fournit au mieux une approximation de l'interface sigvec BSD.  

CONFORMITÉ

BSD, SVr4.  

VOIR AUSSI

sigaction(2), signal(2)  

TRADUCTION

Christophe Blaess, 1996-2003.
