NOM

locale.alias - Base de données des alias des locales  

DESCRIPTION

La base de données locale.alias (fichier /etc/locale.alias) est utilisée par la commande locale et le système X Window Ce fichier décrit les alias pour les locales, chaque ligne étant de la forme :
  <alias> <nom-de-la-locale>

où <nom-de-la-locale> est au format POSIX : xx_YY.CHARSET. Les deux premières lettres xx représentent le code ISO-639 de la langue,
 les deux suivantes YY le code ISO-3166 du pays,  et CHARSET représente un des jeux de caractères (listés dans /usr/share/i18n/charsets ). La définition des alias est libre ; ils sont généralement le nom de la langue en anglais, ou plus simplement le nom POSIX de la locale.

Les lignes commençant par dièse ("#") sont considérés comme des commentaires et sont ignorées.
