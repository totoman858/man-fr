NOM

clog2, clog2f, clog2l - Logarithmes binaires de nombres complexes.  

SYNOPSIS

#include <complex.h>

double complex clog2(double complexz);

float complex clog2f(float complex z);

long double complex clog2l(long double complexz);

 

DESCRIPTION

Le logarithme binaire est défini ainsi : log2(cabs(z))+i*carg(z). Notez que si z est très proche de zéro, un débordement se produira.  

CONFORMITÉ

C99  

VOIR AUSSI

cabs(3), cexp(3), clog(3), clog10(3)  

TRADUCTION

Christophe Blaess, 2003.
