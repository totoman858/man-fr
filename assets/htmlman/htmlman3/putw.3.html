NOM

getw, putw - Lecture et écriture de mots (ints).  

SYNOPSIS

#include <stdio.h>

int getw (FILE * stream);
int putw (int w, FILE * stream);
 

DESCRIPTION

getw lit un mot (c'est à dire un entier int) depuis le flux stream. Cette fonction est fournie pour des raisons de compatibilité avec SVID. Nous recommandons l'utilisation de fread(3) à sa place.

putw écrit un mot (c'est à dire un entier int) dans le flux stream. Cette fonction est fournie pour des raisons de compatibilité avec SVID. Nous recommandons l'utilisation de fwrite(3) à sa place.  

VALEUR RENVOYÉE

Normalement, getw renvoie le mot lu, et putw renvoie 0. En cas d'erreur ils renvoient EOF.  

BOGUES

La valeur renvoyée en cas d'erreur est aussi une valeur légitime pour une donnée. On peut utiliser ferror(3) pour distinguer ces cas.  

CONFORMITÉ

SVID  

VOIR AUSSI

fread(3), fwrite(3), ferror(3), getc(3), putc(3)
