NOM

exp2 - Fonction exponentielle base 2.  

SYNOPSIS

#include <math.h>

double exp2(double x);

float exp2f(float x);

long double exp2l(long double x);
 

DESCRIPTION

La fonction exp2() renvoie la valeur de 2 élevé à la puissance x.  

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899. Les variantes float et long double sont des demandes C99.  

VOIR AUSSI

exp(3), exp10(3), log2(2)
