NOM

defoma-hints - fabrique les fichiers d'indications de polices.  

SYNOPSIS

defoma-hints [-c] catégorie polices...
 

DESCRIPTION

defoma-hints est l'outil qui fabrique les fichiers d'indications de polices. Il pose diverses questions au sujet des types de caractères, extrait des données des fichiers de polices et affiche les indications sur la sortie standard, au format des fichiers d'indications.  

FICHIERS

/usr/share/defoma/libhint-*.pl
Bibliothèques Perl spécifiques à chaque catégorie de police, elles fabriquent les fichiers d'indications de cette catégorie.
 

EXEMPLES

1. Création du fichier d'indications type1.hints pour les polices de catégorie type1 contenues dans le répertoire /usr/share/fonts/type1. 

defoma-hints type1 /usr/share/fonts/type1/*.pfb > type1.hints
