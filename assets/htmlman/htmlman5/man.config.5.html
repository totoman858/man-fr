NOM

man.config - fichier de configuration pour man  

DESCRIPTION

Ce fichier est lu par man(1) et contient (a) l'emplacement des pages de manuel, (b) l'emplacement absolu de divers programmes comme nroff, eqn, tbl etc. utilisés par man, et (c) une liste de décompresseurs pour les fichiers possédant l'extension indiquée. Un autre fichier de configuration peut être indiqué à l'aide de l'option

man -C private_man.config ...

Les noms de commandes peuvent être fournis avec des options. Les options utiles de nroff peuvent être trouvées dans grotty(1). Par exemple, au lieu de la ligne proposée par défaut 

NROFF /usr/bin/groff -mandoc -Tlatin1

il est possible d'écrire 

NROFF /usr/bin/groff -mandoc -Tlatin1 -P-u -P-b

afin de supprimer le soulignement et les mots barrés.  

BOGUES / AVERTISSEMENTS

[ Note du traducteur ]
Si vous lisez cette page, vous êtes probablement intéressé par la documentation en français. Afin que man affiche correctement les caractères accentués, il vous faut changer les options -Tascii par -Tlatin1 en particulier dans /etc/man.config ou /usr/lib/man.config.
