NOM

defoma-reconfigure - configure à nouveau les polices.  

SYNOPSIS

defoma-reconfigure [-f]  

DESCRIPTION

defoma-reconfigure configure à nouveau les polices, en reprenant tout à zéro. Ceci résout un bogue critique de perte de données et permet aussi de tout configurer.

defoma-reconfigure supprime d'abord presque tous les fichiers de /var/lib/defoma pour effacer toutes les informations de configuration des polices, puis lance defoma-app en lui demandant une mise à jour (commande update). Si vous avez vous-même placé un fichier important dans /var/lib/defoma, vous devriez sauvegarder ce répertoire avant de lancer defoma-reconfigure.

L'option -f est utilisée pour corriger la corruption de données produite par l'interruption brutale de defoma-app.  

VOIR AUSSI

defoma(1).
