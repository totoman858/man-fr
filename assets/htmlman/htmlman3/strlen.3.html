NOM

strlen - Calculer la longueur d'une chaîne de caractères.  

SYNOPSIS

#include <string.h>

size_t strlen (const char *s);
 

DESCRIPTION

La fonction strlen() calcule la longueur de la chaîne de caractères s, sans compter le carèctere nul `\0' final.  

VALEUR RENVOYÉE

La fonction strlen() renvoie le nombre de caractères dans la chaîne s  

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899  

VOIR AUSSI

string(3)
