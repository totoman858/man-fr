NOM

wmemset - Remplir un tableau de caractères larges avec un caractère large constant.  

SYNOPSIS

#include <wchar.h>

wchar_t *wmemset (wchar_t *wcs, wchar_t wc, size_t n);
 

DESCRIPTION

La fonction wmemset est l'équivalent pour les caractères larges de la fonction memset. Elle remplit un tableau de n caractères larges commencant à l'adresse wcs avec n copies du caractère large wc.  

VALEUR RENVOYÉE

wmemset renvoie wcs.  

CONFORME A

ISO/ANSI C, UNIX98  

VOIR AUSSI
