NOM

towctrans - Transformation de caractères larges.  

SYNOPSIS

#include <wctype.h>

wint_t towctrans (wint_t wc, wctrans_t desc);
 

DESCRIPTION

Si wc est un caractère large, la fonction towctrans le traduit en fonction du descripteur de transformation desc. Si wc est WEOF, cette fonction renvoie WEOF.

desc doit être un descripteur de transformation renvoyé par la fonction wctrans.  

VALEUR RENVOYÉE

La fonction towctrans renvoie le caractère transformé, ou WEOF si wc est WEOF.  

CONFORMITÉ

ISO/ANSI C, UNIX98  

VOIR AUSSI

wctrans(3), towlower(3), towupper(3)  

NOTES

Le comportement de la fonction towctrans dépend de la catégorie LC_CTYPE de la localisation.
