NOM

dirfd - Obtenir un descripteur fichier pour un répertoire.  

SYNOPSIS

#include <sys/types.h>
#include <dirent.h>

int dirfd(DIR *rep);  

DESCRIPTION

La fonction dirfd() renvoie le descripteur de fichier associé au flux de répertoire rep.  

ERRORS

En cas d'erreur, -1 est renvoyée.  

NOTES

Le prototype de dirfd n'est disponible que si _BSD_SOURCE ou _SVID_SOURCE est définie (de manière explicite, ou implicite, en ne définissant pas _POSIX_SOURCE ou en compilant avec l'option -ansi).  

CONFORMITÉ

Ceci est une extension BSD, présente dans 4.3BSD-Reno, mais pas dans 4.2BSD. Elle est également présente dans la libc5 (depuis la version 5.1.2) et dans la GlibC2.
