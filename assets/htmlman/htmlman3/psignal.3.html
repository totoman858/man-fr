NOM

psignal - Afficher le libellé d'un signal.  

SYNOPSIS

#include <signal.h>

void psignal (int sig, const char *s);

extern const char *const sys_siglist[];
 

DESCRIPTION

La fonction psignal() affiche un message sur stderr constitué de la chaîne s, d'un deux-points, d'un espace, et d'un libellé décrivant le signal numéro sig. Si sig est invalide, le message affiché indiquera un signal inconnu.

La table sys_siglist contient les libellés des messages, indexés par le numéro de signal.  

VALEUR RENVOYÉE

La fonction psignal() ne renvoie aucune valeur.  

CONFORMITÉ

BSD 4.3  

VOIR AUSSI

perror(3), strsignal(3)
