NOM

true - Réussir à ne rien faire...  

SYNOPSIS

true
true {--help,--version}  

DESCRIPTION

Cette page de manuel documente la version GNU de true (et probablement toutes les autres versions également).

true ne fait rien d'autre que renvoyer un code de retour valant 0, signifiant ``réussite''.

Il peut être utilisé pour remplir un shell script lorsqu'une commande réussie est nécessaire, bien que la commande shell intégrée `:' (double-point) fasse la même chose, plus vite.  

OPTIONS

Quand la version GNU de true est invoquée avec un seul argument, les options suivantes sont reconnues :
--help
Afficher un message d'aide sur la sortie standard et se terminer normalement.
--version
Afficher un numéro de version sur la sortie standard et se terminer normalement.
