NOM

asinh, asinhf, asinhl - Fonction Argument Sinus Hyperbolique.  

SYNOPSIS

#include <math.h>

double asinh (double x);
float asinhf (float x);
long double asinhl (long double x);
 

DESCRIPTION

La fonction asinh() calcule l'Argument Sinus Hyperbolique de x, c'est à dire la valeur dont le Sinus Hyperbolique est x. Le domaine de définition de cette fonction étant l'ensemble des réels, elle ne renvoie jamais d'erreur.  

NOTES

Pour utiliser les fonctions de la bibliothèque mathématique, il ne faut pas oublier d'ajouter l'argument -lm sur la ligne de commande de cc(1) ou ld(1), lors de l'édition des liens.

[NDT] : Pour mémoire, mathématiquement (je ne connais pas l'implémentation), l'argument sinus hyperbolique d'une variable x réelle :

ArgSinH (x) = ln ( x + (x^2 + 1) ^ (1/2) )

 

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899. Les variantes float et long double sont des demandes C99.  

VOIR AUSSI

acosh(3), atanh(3), cosh(3), sinh(3), tanh(3)
