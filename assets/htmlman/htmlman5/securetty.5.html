NOM

securetty - Liste de terminaux sécurisés.  

DESCRIPTION

/etc/securetty est utilisé par login(1).

Le fichier contient une liste de terminaux (tty sans le préfixe /dev/) sur lesquels la connexion de 'root' est autorisée. Voir login.defs(5) si vous utilisez le mécanisme de mots-de-passe shadow.  

FICHIERS

/etc/securetty  

VOIR AUSSI

login(1), login.defs(5)
