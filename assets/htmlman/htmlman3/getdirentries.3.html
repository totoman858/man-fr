GETDIRENTRIES

NOM

getdirentries - Lire le contenu d'un répertoire dans un format indépendant du système.  

SYNOPSIS

#define _BSD_SOURCE or #define _SVID_SOURCE
#include <dirent.h>

ssize_t getdirentries (int fd, char *buf, size_t nbytes , off_t *basep);  

DESCRIPTION

La fonction getdirentries lit les entrées d'un répertoire indiqué par fd dans le buffer buf. nbytes octets au maximum sont lus. La lecture démarre à la position *base, puis *basep est mis à jour avec la nouvelle position à lire.  

VALEUR RENVOYÉE

getdirentries renvoie le nombre d'octets lus, ou 0 à la fin du répertoire. En cas d'échec, -1 est renvoyé, et errno contient le code d'erreur.
