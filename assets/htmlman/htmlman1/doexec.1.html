NOM

doexec - exécute un programme avec un argv[0] forcé  

SYNOPSIS

doexec /chemin/de/l'exécutable argv[0] [argv[1-n]]  

DESCRIPTION

doexec se contente de lancer le programme indiqué avec la liste argv fournie. Cela permet de forcer un argv[0] différent du nom du programme.  

OPTIONS

Toutes les options sont passées dans la liste argv au programme à exécuter.
