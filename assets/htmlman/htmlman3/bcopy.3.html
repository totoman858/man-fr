NOM

bcopy - Copier un bloc d'octets.  

SYNOPSIS

#include <string.h>

void bcopy (const void *src, void *dest, size_t n);
 

DESCRIPTION

La fonction bcopy() copie n octets depuis src vers dest. Le résultat est correct même si les deux zones se chevauches.  

VALEUR RENVOYÉE

Aucune.  

CONFORMITÉ

BSD 4.3. Cette fonction est déconseillée, utilisez plutôt memcpy dans les nouveaux programmes. Notez que les deux premiers paramètes sont inversés par rapport à memcpy.  

VOIR AUSSI

memccpy(3), memcpy(3), memmove(3), strcpy(3), strncpy(3)
