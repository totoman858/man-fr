NOM

bcmp - Comparaison de blocs d'octets.  

SYNOPSIS

#include <string.h>

int bcmp (const void *s1, const void *s2, size_t n);
 

DESCRIPTION

La fonction bcmp () compare les blocs de n octets s1 et s2. S'ils sont égaux, et en particulier si n est nul, bcmp() renvoie zéro. Sinon une valeur non-nulle est renvoyée.  

VALEUR RENVOYÉE

La fonction bcmp() renvoie 0 si les deux blocs sont égaux, et une valeur non-nulle sinon.  

CONFORMITÉ

BSD 4.3. Cette fonction est déconseillée, utilisez plutôt memcmp pour de nouveaux programmes.  

VOIR AUSSI

memcmp(3), strcasecmp(3), strcmp(3), strcoll(3), strncmp(3), strncasecmp(3)
