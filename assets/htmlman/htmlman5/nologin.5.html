NOM

nologin - Empêcher les utilisateurs de se connecter au système.  

DESCRIPTION

/etc/nologin est utilisé par login(1). Si le fichier existe, la connexion ne sera autorisée que pour le Super-User. Les autres utilisateurs verront s'afficher le contenu du fichier (généralement un message comme "Maintenance système en cours") et la connexion sera refusée.  

EXEMPLE

# ls -l /etc/nologin
-rw-r--r--  1 root  root     51 Oct 17 17:15 /etc/nologin
# cat /etc/nologin

Système indisponible jusqu'à 09:30 (maintenance)

#
 

FICHIER

/etc/nologin  

VOIR AUSSI

login(1), shutdown(8)
