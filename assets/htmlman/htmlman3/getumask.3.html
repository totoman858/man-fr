GETUMASK

NOM

getumask - Renvoyer le masque de permission pour la création de fichier.  

SYNOPSIS

#include <sys/types.h>
#include <sys/stat.h>

mode_t getumask(void);  

DESCRIPTION

Cette fonction renvoie le masque de permissions utilisé lors de la création d'un fichier. Elle est équivalente à 
mode_t getumask(void) {
        mode_t mask = umask(0);
        umask(mask);
        return mask;
}
excepté qu'elle est explicitement prévue pour pouvoir être utilisée de manière saine dans le contexte d'une application multi-threads (elle partage un sémaphore avec la fonction umask()).  

NOTES

Cette fonction est documentée mais non encore implémentée dans la GlibC version 2.2.5.  

CONFORMITÉ

Ceci est une extension GNU peu utilisée.  

VOIR AUSSI

umask(2)
