NOM

pppoe-start - Script shell pour établir un lien PPPoE.  

SYNOPSIS

pppoe-start [fichier_config]

pppoe-start interface utilisateur [fichier_config]

 

DESCRIPTION

pppoe-start est un script shell qui lance le client PPPoE espace utilisateur de Roaring Penguin. Si vous omettez fichier_config, le fichier par défaut /etc/ppp/pppoe.conf est utilisé. Si vous fournissez interface et utilisateur, alors ils auront priorité sur l'interface Ethernet et le nom d'utilisateur établis dans le fichier de configuration.
