NOM

ptsname - Obtenir le nom d'un pseudo-terminal esclave.  

SYNOPSIS

#include <stdlib.h>

char *ptsname(int fd);
 

DESCRIPTION

La fonction ptsname() renvoie le nom du périphérique pseudo-terminaml esclave (pty) associé au pseudo-terminal maître indiqué par fd.  

VALEUR RENVOYÉE

Lorsqu'elle réussit, la fonction ptsname() renvoie un pointeur sur une chaîne de stockage statique. Le pointeur ne doit PAS être libéré.

Si elle échoue, ptsname() renvoie un pointeur NULL.  

CONFORMITÉ

POSIX 1003.1-2001. Cette fonction fait partie du support pty d'Unix 98. Voir pts(4).  

VOIR AUSSI

grantpt(3), ttyname(3), pts(4)
