NOM

apt-sortpkgs - Utilitaire pour trier des fichiers d'index de paquet  

SYNOPSIS

apt-sortpkgs [ -hvs ] [ -o=config string ] [ -c=file ] file...  

DESCRIPTION

apt-sortpkgs prendra un fichier index (source ou paquet) and triera les enregistrements de telle façon qu'ils soient ordonnés par nom. Il triera aussi les champs internes de chaque enregistrement selon les règles de tri internes.

Toute sortie est envoyée vers la sortie standard (stdout), l'entrée doit être un fichier "seekable".  

OPTIONS

Toutes les options de la ligne de commande peuvent être positionnées via le fichier de configuration, les descriptions indiquent les options de configuration à valider. Pour les options de type booléen, on peut passer outre le fichier de configuration en utilisant quelque chose comme -f-,--no-f, -f=no ou d'autres variantes.

-s
--source <source>
Utilise l'ordonnancement du champ index de <source>. Configuration: APT::SortPkgs::Source.
-h
--help
Montre un court résumé de l'utilisation.
-v
--version
Montre la version du programme.
-c
--config-file <fichier de configuration>
Specifie un fichier de configuration à utiliser. Le programme lira le fichier de configuration par défaut puis celui-ci. Cf. apt.conf(5) pour des informations liées a la syntaxe.
-o
--option
Positionne une option de configuration. Positionnera une option de configuration de facon arbitraire. La syntaxe est -o Foo::Bar=bar.
 

VOIR AUSSI

apt.conf(5)  

DIAGNOSTICS

apt-sortpkgs retourne zéro pour une opération normale, la valeur décimale de 100 en cas d'erreur.
