NOM

users - Afficher le nom des utilisateurs actuellement connectés.  

SYNOPSIS

users [FICHIER]
users {--help,--version}  

DESCRIPTION

Cette page de manuel documente la version GNU de users.

users affiche, sur une seule ligne, la liste des noms des utilisateurs actuellement connectés sur la machine.

Les noms sont séparés par des espaces. Chaque nom affiche correspond à une session de connexion, ainsi si un utilisateur dispose de plusieurs connexions simultanées, son nom apparaîtra autant de fois.

Sans arguments, users lit ses informations depuis le fichier /etc/utmp . Si un argument FICHIER est indiqué, users lit ce fichier à la place. Un choix classique de FICHIER est /etc/wtmp .  

OPTIONS

--help
Afficher un message d'aide sur la sortie standard et se terminer normalement.
--version
Afficher un numéro de version sur la sortie standard et se terminer normalement.
