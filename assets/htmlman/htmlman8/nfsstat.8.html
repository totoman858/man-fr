NOM

nfsstat - Afficher des statistiques NFS.  

SYNOPSIS

/usr/sbin/nfsstat [-anrcsz] [-o domaine] ...  

DESCRIPTION

La commande nfsstat affiche des statistiques sur l'état du sous-système NFS du noyau. Actuellement, seules les statistiques du serveur peuvent être affichées car le client NFS ne fait pas encore de statistiques [Ndt: hum...]

 

OPTIONS

-s
Affiche seulement les informations sur le serveur. Par défaut, les informations concernant et le serveur et le client sont affichées.
-c
Affiche seulement les informations sur le client.
-n
Affiche seulement les statitiques sur NFS. Par défaut, les informations sur RPC et NFS sont affichées.
-r
Affiche seulement les statistiques sur les RPC.
-z
Réinitialise les compteurs du noyau. Option non supportée.
-o domaine
Affiche des statistiques au sujet du domaine indiqué. Ce dernier peut être l'un des suivants :
nfs
Informations sur le protocole NFS, selon les appels RPC.
rpc
Information générales sur les RPC.
net
Statistiques de la pile réseau telles que nombre de paquets reçus, nombre de connections TCP,...
fh
Informations sur l'utilisation du cache de descripteurs de fichier du serveur, en particulier le nombre total de lookups, et le nombre de réponses trouvées dans le cache etcelui de celles non cachées.
rc
Information sur l'utilisation du cache des réponses aux requêtes, y compris le nombre total de lookups, et le nombre de réponses trouvées dans le cache et celui de celles non cachées.
 

EXEMPLES

 

FICHIERS

/proc/net/rpc/nfsd
Interface basée sur procfs pour les statistiques du serveur NFS implémenté dans le noyau.
/proc/net/rpc/nfs
Interface basée sur procfs pour les statistiques du client NFS implémenté dans le noyau.
