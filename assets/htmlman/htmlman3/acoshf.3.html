NOM

acosh, acoshf, acoshl - Fonction Argument Cosinus Hyperbolique.  

SYNOPSIS

#include <math.h>

double acosh (double x);
float acoshf (float x);
long double acoshl (long double x);
 

DESCRIPTION

La fonction acosh() calcule l'Argument Cosinus Hyperbolique de x, c'est à dire la valeur dont le Cosinus Hyperbolique est x. Si x est inférieur à 1.0, acosh() renverra la valeur NaN (not-a-number), et errno contiendra le code d'erreur EDOM.  

ERREURS

EDOM
x est inférieur à 1.0.
 

NOTES

Pour utiliser les fonctions de la bibliothèque mathématique, il ne faut pas oublier d'ajouter l'argument -lm sur la ligne de commande de cc(1) ou ld(1), lors de l'édition des liens.

[NDT] : Pour mémoire, mathématiquement (je ne connais pas l'implémentation), l'argument cosinus hyperbolique d'une variable x, supérieure à 1 vaut :

ArgCosH (x) = ln ( x + (x^2 - 1) ^ (1/2) )

 

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899. Les variantes float et long double sont des demandes C99.  

VOIR AUSSI

asinh(3), atanh(3), cosh(3), sinh(3), tanh(3)
