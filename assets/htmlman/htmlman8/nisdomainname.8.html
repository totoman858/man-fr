NOM

domainname - affiche ou définit le nom de domaine du système
nisdomainname - affiche ou définit le nom de domaine NIS(YP) du système
ypdomainname - affiche ou définit le nom de domaine NIS(YP) du système  

SYNOPSYS

domainname [ domaine ]
nisdomainname [ domaine ]
ypdomainname [ domaine ]

 

DESCRIPTION

Appelé sans argument, domainname fournit le nom de domaine NIS du système. domainname utilise la fonction getdomainname(2) pour obtenir le nom de domaine, nisdomainname et ypdomainname utilisent yp_get_default_domain(3).

Seul le superutilisateur peut définir le nom du domaine en spécifiant le nouveau domaine comme argument. Les trois programmes utilisent setdomainname(2) pour cela. Notons que les programmes tournant déjà risquent de continuer à utiliser l'ancien nom de domaine.  

VOIR AUSSI

getdomainname(2), setdomainname(2), yp_get_default_domain(3), ypbind(8), ypcat(1), ypmatch(1), ypwhich(1), yppoll(8), ypserv(8), yset(8)
