FMOD

NOM

fmod, fmodf, fmodl - Fonction modulo reel.  

SYNOPSIS

#include <math.h>

double fmod (double x, double y);

float fmod (float x, float y);

long double fmod (long double x, long double y);
 

DESCRIPTION

La fonction fmod() calcule le reste de la dvision de x par y. La valeur renvoyée est x - n * y, ou n est le quotient x / y, arrondi à l'entier inférieur en valeur absolue.  

VALEUR RENVOYÉE

La fonction fmod() renvoie le reste décrit ci-dessus, a moins que y soit nul, auquel cas la fonction échoue et errno contient le code d'erreur.  

ERREURS

EDOM
Le dénominateur y est nul.
 

NOTES

Pour utiliser les fonctions de la bibliothèque mathématique, il ne faut pas oublier d'ajouter l'argument -lm sur la ligne de commande de cc(1) ou ld(1), lors de l'édition des liens.  

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899. Les variantes float et long double sont des demandes C99.  

VOIR AUSSI
