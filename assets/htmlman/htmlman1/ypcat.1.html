NOM

ypcat - affiche les valeurs de toutes les clés d'une base de données NIS  

SYNOPSYS

ypcat [ -kt ] [ -d domaine ] [ -h nomhote ] nomcarte
ypcat -x

 

DESCRIPTION

ypcat affiche les valeurs de toutes les clés de la base de données NIS spécifiée par nomcarte, qui peut être un nom ou un surnom de carte.  

OPTIONS

-d domaine
Spécifie un domaine autre que celui par défaut retourné par domainname(1).
-h nomhote
Spécifie un nom d'hôte autre que celui par défaut déterminé par ypbind(8).
-k
Affiche les clés d'une carte. Cette option est utile pour les cartes dans lesquelles les valeurs sont nulles ou pour lesquelles la clé ne fait pas partie de la valeur.
-t
Cette option désactive la traduction des surnoms de cartes.
-x
Affiche la table de traduction des surnoms de carte.
 

FICHIERS

/var/yp/nicknames
table de traduction des surnoms de cartes.
 

VOIR AUSSI

domainname(8), nicknames(5), ypbind(8), ypmatch(1), yppoll(8), ypserv(8), yset(8), ypwhich(1)
