NOM

zgrep - rechercher une expression rationnelle dans des fichiers pouvant être compactés.  

SYNOPSIS

zgrep [ options_grep ] [ -e ] motif nom-fichier...  

DESCRIPTION

zgrep est utilisé pour invoquer la commande grep sur des fichiers compactés par compress ou par gzip. Toutes les options spécifiées sont passées directement à grep. Si aucun fichier n'est spécifié, alors l'entrée standard est décompactée si nécessaire et est envoyée à grep. Sinon, les fichiers fournis sont décompactés si nécessaire et sont envoyés à grep.

Si zgrep est invoqué en tant que zegrep ou zfgrep, alors egrep ou fgrep est utilisé au lieu de grep. Si la variable d'environnement GREP est définie, zgrep l'utilise comme le programme grep à invoquer. Par exemple,


    pour sh :  GREP=fgrep  zgrep chaîne_caract fichiers
    pour csh : (setenv GREP fgrep; zgrep chaîne_caract fichiers)
