NOM

localeconv - Obtenir des informations sur les formats numeriques.  

SYNOPSIS

#include <locale.h>

struct lconv *localeconv (void);
 

DESCRIPTION

La fonction localeconv() renvoie un pointeur sur une struct lconv conforme à la localisation en cours. Cette structure est définie dans le fichier locale.h qui contient toutes les valeurs associées aux catégories de localisation LC_NUMERIC et LC_MONETARY. Les applications peuvent également utiliser les fonctions printf() et strfmon() dont les comportements sont en accord avec la localisation en cours.  

CONFORMITÉ

ANSI C, POSIX.1  

BOGUES

La famille de fonctions printf() peut honorer ou non la localisation.  

VOIR AUSSI

locale(1), localedef(1), strcoll(3), isalpha(3), setlocale(3), strfmon(3), locale(7).
