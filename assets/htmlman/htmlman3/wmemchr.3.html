NOM

wmemchr - Rechercher un caractère large dans une zone de mémoire.  

SYNOPSIS

#include <wchar.h>

wchar_t *wmemchr (const wchar_t *s, wchar_t c, size_t n);
 

DESCRIPTION

La fonction wmemchr est l'équivalent pour les caractères larges de la fonction memchr. Elle recherche la première occurence du caractère large c dans la table de n caractères larges débutant en s.  

VALEUR RENVOYÉE

La fonction wmemchr renvoie un pointeur sur la première occurrence de c au sein de la table de n caractères larges commençant en s, ou NULL si c n'y figure pas.  

CONFORMITÉ

ISO/ANSI C, UNIX98  

VOIR AUSSI

memchr(3), wcschr(3)
