NOM

wcslen - Déterminer la longueur d'une chaîne de caractères larges.  

SYNOPSIS

#include <wchar.h>

size_t wcslen (const wchar_t *s);
 

DESCRIPTION

La fonction wcslen est l'équivalent de la fonction strlen() pour les caractères larges. Elle détermine la longueur de la chaîne pointée par s, sans compter le caractère nul final.  

VALEUR RENVOYÉE

La fonction wcslen renvoie le nombre de caractères larges de s [NdT: pas le nombre d'octets !]  

CONFORMITÉ

ISO/ANSI C, UNIX98  

VOIR AUSSI

strlen(3)
