NOM

cabs, cabsf, cabsl - Modules de nombres complexes.  

SYNOPSIS

#include <complex.h>

double cabs(double complex z);

float cabsf(float double complex z);

long double cabsl(long double complex z);

 

DESCRIPTION

Les fonctions de la famille cabs() renvoient le module du nombre complexe z, ce que certains appellent aussi « valeur absolue Â» de z.

Le résultat est un nombre réel.  

NOTE

Cette est en fait un alias pour hypot(a,b) = sqrt(a*a+b*b).  

CONFORMITÉ

C99  

VOIR AUSSI

abs(3), hypot(3), cimag(3), complex(5)
