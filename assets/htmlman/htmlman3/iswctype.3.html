NOM

iswctype - Classification des caractères larges.  

SYNOPSIS

#include <wctype.h>

int iswctype (wint_t wc, wctype_t desc);
 

DESCRIPTION

Si wc est un caractère large ayant les propriétés décrites par desc (ou autrement dit, appartient à la classe de caractères indiquée par desc), la fonction iswctype renvoie une valeur non-nulle. Sinon elle renvoie zero. Si wc est WEOF, la fonction renvoie zéro.

desc doit être un descripteur de propriété renvoyé par la fonction wctype.  

VALEUR RENVOYÉE

La fonction iswctype renvoie une valeur non-nulle si wc a les propriétés indiquées. Sinon elle renvoie zéro.  

CONFORMITÉ

ISO/ANSI C, UNIX98  

VOIR AUSSI

wctype(3), iswalnum(3), iswalpha(3), iswblank(3), iswcntrl(3), iswdigit(3), iswgraph(3), iswlower(3), iswprint(3), iswpunct(3), iswspace(3), iswupper(3), iswxdigit(3)  

NOTES

Le comportement de la fonction iswctype dépend de la catégorie LC_CTYPE de la localisation.
