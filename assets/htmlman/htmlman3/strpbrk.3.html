NOM

strpbrk - Rechercher un ensemble de caractères dans une chaîne.  

SYNOPSIS

#include <string.h>

char *strpbrk (const char *s, const char *accept);
 

DESCRIPTION

La fonction strpbrk() recherche la première occurence dans la chaîne s de l'un quelconque des caractères de la chaîne accept.  

VALEUR RENVOYÉE

La fonction strpbrk() renvoie un pointeur sur le caractère de s qui corresponde à un caractère de accept, ou NULL si aucune correspondance n'est trouvée.  

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899
