NOM

cbrt, cbrtf, cbrtl - Fonctions racines cubiques  

SYNOPSIS

#include <math.h>

double cbrt(double x);

float cbrtf(float x);

long double cbrtl(long double x);
 

DESCRIPTION

La fonction cbrt() renvoie la racine cubique (réelle) de x. La fonction ne peut pas échouer ; chaque valeur réelle possible a une racine cubique calculable.  

CONFORMITÉ

cbrt() était une extension GNU. C'est devenu une obligation C99.  

VOIR AUSSI

sqrt(3), pow(3)  

TRADUCTION
