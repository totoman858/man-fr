NOM

lpc - Programme de contrôle de l'imprimante  

SYNOPSIS

lpc [ commande [ paramètre(s) ] ]  

DESCRIPTION

lpc permet de contrôler partiellement des imprimantes et classes d'imprimantes de CUPS. Il est également utilisé pour récupérer l'état des files d'attente.

Si aucune option n'est donnée dans la ligne de commande, lpc affichera une invite et prendra ses commandes depuis l'entrée standard.  

COMMANDES

lpc reconnaît un sous-ensemble des commandes reconnues le programme lpc de Berkeley :
exit

Quitte l'interpréteur de commandes.
help [commande]

Affiche un message d'aide.
quit

Quitte l'interpréteur de commandes.
status [file-attente]

Affiche l'état des files d'attente d'une ou plusieurs classes ou imprimantes.
? [commande]

Affiche un message d'aide.
 

LIMITATIONS

Étant donné que lpc est spécifique au système d'impression de Berkeley, il est impossible de l'utiliser pour configurer les files d'attente de CUPS. Pour cela, vous devez utiliser le programme lpadmin(8) ou tout autre client compatible avec CUPS remplissant ces fonctions.  

COMPATIBILITÉ

La version CUPS de lpc n'implémente pas toutes les commandes de la version Berkeley.
