NOM

grpck - vérifie l'intégrité des fichiers de groupes

 

SYNOPSIS

grpck [-rs] [group shadow]

 

DESCRIPTION

grpck vérifie l'intégrité des informations d'authentification du système. Toutes les entrées des fichiers /etc/group et /etc/gshadow sont vérifiées pour voir si elles ont un format correct et si les données de chaque champ sont valides. Il sera demandé à l'utilisateur d'effacer les entrées qui ne sont pas correctement formatées, ou s'il y a des erreurs incorrigibles.

Les vérifications sont faites pour être certain que chaque entrée possède :

- un nombre correct de champs
- un nom de groupe unique
- une liste valide de membres et d'administrateurs

Les vérifications sur le nombre correct de champs et sur l'unicité du nom de groupe sont fatales. Si l'entrée n'a pas un nombre correct de champs, il sera demandé à l'utilisateur d'effacer la ligne complète. Si l'utilisateur ne répond pas par l'affirmative, toutes les autres vérifications ne seront pas effectuées. Dans le cas d'une entrée avec un nom de groupe dupliqué, l'effacement sera proposé à l'utilisateur, mais les vérifications restantes seront effectuées. Toutes les autres erreurs seront indiquées sous forme d'avertissement et l'utilisateur se voit encouragé à lancer la commande groupmod pour corriger les erreurs.

Les commandes qui opèrent sur le fichier /etc/group ne sont pas capables d'altérer les entrées corrompues ou dupliquées. Dans certaines circonstances, grpck devrait être utilisé pour effacer l'entrée fautive.

 

OPTIONS

Par défaut, grpck opère sur les fichiers /etc/group et /etc/gshadow. L'utilisateur peut choisir des fichiers alternatifs avec les paramètres group et shadow. De plus, l'utilisateur peut exécuter la commande en mode lecture seule en utilisant l'option -r. Cette option fait qu'il sera répondu non à toutes les questions relatives à des modifications, sans intervention de l'utilisateur. grpck peut aussi classer les entrées dans les fichiers /etc/group et /etc/gshadow par GID. Pour lancer la commande avec le mode de tri, passez-lui l'option -s. Aucune vérification n'est effectuée, cela effectue simplement le tri.

 

FICHIERS

/etc/group - informations sur les comptes de groupes
/etc/gshadow - mots de passe masqués et informations de l'administrateur de groupe
/etc/passwd - informations sur les utilisateurs

 

VOIR AUSSI

group(5), passwd(5), shadow(5), groupmod(8)

 

DIAGNOSTICS

La commande grpck renvoie les valeurs suivantes en quittant :
0
Succès
1
Erreur de syntaxe
2
Une ou plusieurs mauvaises entrées de groupe
3
Impossible d'ouvrir les fichiers de groupes
4
Impossible de verrouiller les fichiers de groupes
5
Impossible de mettre à jour les fichiers de groupes
