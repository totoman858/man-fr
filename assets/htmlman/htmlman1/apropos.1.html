NOM

apropos - recherche de chaînes de caractères dans la base de données whatis  

SYNOPSIS

apropos mot-clé ...  

DESCRIPTION

apropos effectue une recherche de chaînes de caractères dans un ensemble de fichiers de données contenant de brèves descriptions des commandes système, et affiche le résultat sur la sortie standard.
