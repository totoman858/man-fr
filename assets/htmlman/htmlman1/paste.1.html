NOM

paste - Regrouper les lignes de différents fichiers.  

SYNOPSIS

paste [-s] [-d liste_délimiteurs] [--serial] [--delimiters=liste_délimiteurs] [--help] [--version] [fichier...]  

DESCRIPTION

Cette page de manuel documente la version GNU de paste ([NDT] paste = coller).

paste affiche des lignes regroupant les lignes correspondantes de chaque fichier, séparées par des Tabulations, et terminées par un Saut de Ligne (NewLine).

Si aucun fichier n'est indiqué, ou si le nom `-' est mentionné, la lecture se fait sur l'entrée standard.  

OPTIONS

-s, --serial
Regrouper toutes les lignes d'un seul fichier à la fois, plutôt qu'une ligne de chaque fichier.
-d, --delimiters liste_de_délimiteurs
Utiliser successivement les caractères de la liste_de_délimiteurs plutôt que les Tabulations pour séparer les lignes regroupées. Quand liste_de_délimiteurs est épuisée, reprendre au début de la liste.
--help
Afficher un message d'aide sur la sortie standard et se terminer normalement.
--version
Afficher un numéro de version sur la sortie standard et se terminer normalement.
