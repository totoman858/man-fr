defoma-id

Section: User Commands (1)
Updated: 4 mars 2001
Index Return to Main Contents
 

NOM

defoma-id - gère la base de référence du gestionnaire de polices Debian.  

SYNOPSIS

defoma-id [options] list-cache application
defoma-id [options] install application/base-de-référence identifiant police
defoma-id [options] exclude application/base-de-référence identifiant police
defoma-id [options] unset application/base-de-référence identifiant police
defoma-id [options] add-alias application/base-de-référence identifiant police alias
defoma-id [options] remove-alias application/base-de-référence alias police  

DESCRIPTION

defoma-id est conçu pour être appelé par dfontmgr et non en ligne de commande.  

VOIR AUSSI

defoma(1). dfontmgr(1).
