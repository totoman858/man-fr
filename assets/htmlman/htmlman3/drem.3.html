NOM

drem - Reste d'une division en virgule flottante.  

SYNOPSIS

#include <math.h>

double drem (double x, double y);
 

DESCRIPTION

La fonction drem() calcule le reste de la division de x par y. La valeur renvoyée est x - n * y, ou n est le quotient x / y, arrondi à l'entier le plus proche. Si le quotient vaut 1/2, il est arrondi à la valeur paire.  

VALEUR RENVOYÉE

La fonction drem() renvoie le reste, à moins que y soit nul, auquel cas drem() échoue et errno contient le code d'erreur.  

ERREURS

EDOM
Le dénominateur y est nul.
 

CONFORMITÉ

BSD 4.3  

VOIR AUSSI

fmod(3)
