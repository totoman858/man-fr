NOM

swab - Echanger des octets adjacents.  

SYNOPSIS

#include <unistd.h>

void swab (const void * from, void * to, size_t n);
 

DESCRIPTION

La fonction swab() copie n octets de la chaîne pointée par from dans la chaine pointée par to, en échangeant les octets adjacents de rangs pairs et impairs. Cette fonction permet de transmettre des données entre des machines qui utilisent des ordres poids fort/poids faible différents.

Cette fonction ne fait rien si n est négatif. Lorsque n est positif et impair, elle manipule n-1 octets comme décrit plus haut, et agit de manière non précisée avec le dernier octet. Autrement dit, n doit être pair !)  

VALEUR RENVOYÉE

La fonction swab() ne renvoie pas de valeur.  

CONFORMITÉ

SVID 3, BSD 4.3, POSIX 1003.1-2001
