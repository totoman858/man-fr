NOM

wcschr - Rechercher un caractère large dans une chaîne.  

SYNOPSIS

#include <wchar.h>

wchar_t *wcschr (const wchar_t *wcs, wchar_t wc);
 

DESCRIPTION

La fonction wcschr est l'équivalent pour les caractères larges de la fonction strchr. Elle recherche la première occurence de wc dans la chaîne de caractères larges pointée par wcs.  

VALEUR RENVOYÉE

La fonction wcschr renvoie un pointeur sur la première occurence de wc dans la chaîne de caractères larges wcs, ou NULL si wc ne s'y trouve pas.  

CONFORMITÉ

ISO/ANSI C, UNIX98  

VOIR AUSSI

strchr(3), wcsrchr(3), wcspbrk(3), wcsstr(3), wmemchr(3)
