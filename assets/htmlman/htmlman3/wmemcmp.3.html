NOM

wmemcmp - Comparer deux tables de caractères larges.  

SYNOPSIS

#include <wchar.h>

int wmemcmp (const wchar_t *s1, const wchar_t *s2, size_t n);
 

DESCRIPTION

La fonction wmemcmp est l'équivalent pour les caractères larges de la fonction memcmp. Elle compare les n caractères larges commençant en s1 aux n commençant en s2.  

VALEUR RENVOYÉE

La fonction wmemcmp renvoie zéro si les tables de caractères larges, de taille n situées en s1 et s2 sont égales. Elle renvoie un entier positid si à l'emplacement i de la première différence (i < n), le caractère large s1[i] est supérieur à s2[i]. Elle renvoie un entier négatif si s1[i] est inférieur à s2[i].  

CONFORMITÉ

ISO/ANSI C, UNIX98  

VOIR AUSSI

memcmp(3), wcscmp(3)
