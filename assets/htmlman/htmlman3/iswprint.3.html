NOM

iswprint - Vérifier si un caractère large est imprimable.  

SYNOPSIS

#include <wctype.h>

int iswprint (wint_t wc);
 

DESCRIPTION

La fonction iswprint est l'équivalent pour les caractères larges de la fonction isprint. Elle teste si le caractère large wc appartient à la classe de caractères "print".

La classe de caractères "print" est disjointe de la classe "cntrl".

La classe "print" contient la classe "graph".  

VALEUR RENVOYÉE

La fonction iswprint renvoie une valeur non nulle si le caractère large wc appartient à la classe "print". Sinon, 0 est renvoyé.  

CONFORMITÉ

ISO/ANSI C, UNIX98  

VOIR AUSSI

isprint(3), iswctype(3)  

NOTES

Le comportement de iswprint dépend de la valeur de la variable courante de localisation LC_CTYPE.
