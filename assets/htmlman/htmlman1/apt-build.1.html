NOM

apt-build - Rapatrie les sources et construit des paquets optimisés pour votre architecture.  

SYNOPSIS

apt-build <options> [ update ] [ upgrade ] [ world ] [ install pkg ] [ remove pkg ] ] [ info pkg ]  

DESCRIPTION

apt-build est une façade d'apt-get pour construire et installer des paquets optimisés pour une architecture donnée.

 

COMMANDES

update
Rapatrie les listes de nouveaux paquets
upgrade
Réalise une mise à jour
world
Reconstruit votre système
install
Construit et installe de nouveaux paquets
source
Télécharge et décompacte les sources dans le répertoire de construction
info
Information sur un paquet qui peut être construit
remove
Enlève un paquet
dist-upgrade
Mise à jour de la distribution, regardez apt-get(8)
clean-repository
Efface les paquets qui ont été construits
update-repository
Met à jour le référentiel
 

OPTIONS

--help 
Affiche l'aide.
--show-upgraded, -u
Montre les paquets qui seront mis à jour.
--no-wrapper
Ne pas utiliser le wrapper de gcc.
--remove-builddep
Enlève les dépendances de construction installées par apt-build.
--no-source
Ne pas télécharger les sources.
--build-dir
Spécifier le répertoire de construction.
--build-only
Paquet binaire uniquement.
--rebuild
Reconstruit un paquet.
--reinstall
Construit et réinstalle un paquet déjà installé sur votre système.
--build-command <commande>
Utilisez cette commande pour construire un paquet.
--patch <fichier>
Appliquer une rustine avant de construire. Vous pouvez utiliser cette option une ou plusieurs fois.
--patch-strip, -p <nombre>
Préfixe pour appliquer la rustine (0 = -p0, 1 = -p1 ...).
--yes, -y
Répond oui à toutes les questions.
--purge
Purge au lieu d'enlever.
--version, -v
Affiche la version.
