NOM

strsignal - Obtenir un libellé decrivant un signal.  

SYNOPSIS

#include <string.h>

char *strsignal (int sig);

extern const char * const sys_siglist[];
 

DESCRIPTION

La fonction strsignal() renvoie une chaîne décrivant le signal numéro sig. La chaîne ne peut être utilisée que jusqu'à l'appel suivant de strsignal().

La chaîne sys_siglist contient les libellés des signaux, indexés par leurs numéros. Il vaut mieux utiliser strsignal() si possible.  

VALEUR RENVOYÉE

La fonction strsignal() renvoie le libellé approprié, ou un message indiquant un signal inconnu si le numéro est invalide. Sur certains systèmes (mais pas Linux) un pointeur NULL peut être renvoyé en cas de signal inexistant.  

CONFORMITÉ

Cette fonction ne fait partie d'aucun standard mais on la trouve sous Linux et sur certains autres systèmes Unix (comme Solaris).  

VOIR AUSSI

psignal(3), strerror(3)
