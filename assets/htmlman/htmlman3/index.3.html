NOM

index, rindex - Recherche de caractères dans une chaîne.  

SYNOPSIS

#include <string.h>

char *index (const char *s, int c);

char *rindex (const char *s, int c);
 

DESCRIPTION

La fonction index() renvoie un pointeur sur la première occurence du caractère c dans la chaîne s.

La fonction rindex() renvoie un pointeur sur la dernière occurence du caractère c dans la chaîne s.

Le caractère nul final est considéré comme faisant partie de la chaîne.  

VALEUR RENVOYÉE

Les fonctions index() et rindex() renvoient un pointeur sur le caractère correspondant, ou NULL si le caractère n'est pas trouvé.  

CONFORMITÉ

BSD 4.3  

VOIR AUSSI

memchr(3), strchr(3), strpbrk(3), strrchr(3), strsep(3), strspn(3), strstr(3), strtok(3)
