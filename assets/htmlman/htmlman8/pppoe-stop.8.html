NOM

pppoe-stop - Script shell pour fermer un lien PPPoE.  

SYNOPSIS

pppoe-stop [fichier_config]

 

DESCRIPTION

pppoe-stop est un script shell qui ferme le client PPPoE espace utilisateur de Roaring Penguin. Si vous omettez fichier_config, le fichier par défaut /etc/ppp/pppoe.conf est utilisé.
