NOM

fmax - Calculer le maximum.  

SYNOPSIS

#include <math.h>

double fmax(double x, double y);

float fmaxf(float x, float y);

long double fmaxl(long double x, long double y);

 

DESCRIPTION

Renvoie le maximul de x et y.  

CONFORMITÉ

C99  

VOIR AUSSI
