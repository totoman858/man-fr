NOM

wmemmove - Copier un tableau de caractères larges.  

SYNOPSIS

#include <wchar.h>

wchar_t *wmemmove (wchar_t *dest, const wchar_t *src, size_t n);
 

DESCRIPTION

La fonction wmemmove est l'équivalent pour les caractères larges de la fonction memmove. Elle copie n caractères larges depuis le tableau pointé par src dans le tableau pointé par dest. Les tableaux peuvent se recouvrir.

Le programmeur doit s'assurer qu'il y a suffisamment d'espace pour au moins n caractères larges dans dest.  

VALEUR RENVOYÉE

wmemmove renvoie dest.  

CONFORME A

ISO/ANSI C, UNIX98  

VOIR AUSSI

memmove(3), wmemcpy(3)
