NOM

get_thread_area - Obtenir une zone de stockage local pour thread (TLS)  

SYNOPSIS

#include <linux/unistd.h>

#include <asm/ldt.h>

int get_thread_area (struct user_desc *u_info);

 

DESCRIPTION

get_thread_area renvoie une entrée dans la table de stockage local des threads (TLS). L'index de cette entrée correspond à la valeur de u_info->entry_number, transmise par l'utilisateur. Si la valeur est dans les limites correctes, get_thread_info copie l'entrée TLS correspondante dans la zone pointée par u_info.

 

VALEUR RENVOYÉE

get_thread_area renvoie 0 s'il réussit. Sinon il renvoie l'une des erreurs décrites ci-dessous.

 

ERREURS

EINVAL
u_info->entry_number est hors des limites.

EFAULT
u_info est un pointeur invalide

 

CONFORMITÉ

get_thread_area est spécifique à Linux et ne doit pas être employé dans un programme conçu pour être portable.

 

HISTORIQUE

Une version de get_thread_area est apparue dans Linux 2.5 en août 2002.

 

VOIR AUSSI

set_thread_area(2), modify_ldt2.
