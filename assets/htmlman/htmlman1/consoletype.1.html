NOM

consoletype - affiche le type de la console connectée à l'entrée standard  

SYNOPSIS

consoletype  

DESCRIPTION

consoletype affiche le type de la console connectée à l'entrée standard. Il affiche vt si la console est un terminal virtuel (périphérique /dev/tty* ou /dev/console si l'on n'est pas sur une console série) serial si l'entrée standard est une console série (/dev/console ou /dev/ttyS*) et pty si l'entrée standard est un pseudo terminal.  

VALEUR DE RETOUR

consoletype renvoie
0
pour un terminal virtuel
1
pour une console série
2
pour un pseudo terminal.
