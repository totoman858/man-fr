NOM

bzero - Ecrire des zéros dans un bloc d'octets.  

SYNOPSIS

#include <string.h>

void bzero (void *s, size_t n);
 

DESCRIPTION

La fonction bzero() met à 0 les n premiers octets du bloc pointé par s.  

VALEUR RENVOYÉE

La fonction bzero() ne renvoie pas de valeur.  

CONFORMITÉ

BSD 4.3. Cette fonction est déconseillée, utilisez plutôt memset.  

VOIR AUSSI

memset(3), swab(3)
