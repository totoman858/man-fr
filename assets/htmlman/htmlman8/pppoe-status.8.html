NOM

pppoe-status - Script shell pour rapporter l'état d'un lien PPPoE.  

SYNOPSIS

pppoe-status [fichier_config]

 

DESCRIPTION

pppoe-status est un script shell qui contrôle l'état d'un lien PPPoE établi par le client PPPoE espace utilisateur de Roaring Penguin. Si vous omettez fichier_config, le fichier par défaut /etc/ppp/pppoe.conf est utilisé.
