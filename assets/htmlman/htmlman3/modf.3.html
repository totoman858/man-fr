NOM

modf, modff, modfl - Séparer la partie entière et décimale d'un nombre réel.  

SYNOPSIS

#include <math.h>

double modf(double x, double *iptr);

float modf(float x, float *iptr);

long double modf(long double x, long double *iptr);
 

DESCRIPTION

La fonction modf() scinde son argument x en une partie entière et une partie décimale, chacune d'elle ayant le signe de x. La partie entière est stockée dans iptr.  

VALEUR RENVOYÉE

La fonction modf() renvoie la partie décimale de x.  

CONFORMITÉ

SVID 3, POSIX, BSD 4.3, ISO 9899. Les variantes float et long double sont des demandes C99.  

VOIR AUSSI

frexp(3), ldexp(3)  

TRADUCTION

Christophe Blaess, 1996-2003.
