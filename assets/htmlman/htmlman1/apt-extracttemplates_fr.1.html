NOM

apt-extracttemplates - Utilitaire d'extraction des scripts de configuration et fichiers modèles Debconf des paquets Debian  

SYNOPSIS

apt-extracttemplate [ -hv ] [ -t=repertoire temporaire ] fichier...  

DESCRIPTION

apt-extracttemplates reçoit en entrée un ou plusieurs paquets Debian et écrit (dans un répertoire temporaire) tous les scripts de configuration et les fichiers modèles associés. À chaque passage dans un paquet contenant des scripts de configuration et fichiers modèles, une ligne est générée en sortie sous la forme :

paquet version fichier-modèle script-de-configuration

les fichier-modèle et script-de-configuration sont copiés dans un répertoire temporaire désigné par l'option -t ou --tempdir (APT::ExtractTemplates::TempDir>), avec des noms de fichiers sous la forme template.XXXX et config.XXXX  

OPTIONS

Toutes les options de la ligne de commande peuvent être configurées via le fichier de configuration, les descriptions expliquent les configurations possibles de l'option. Pour les options booléennes vous pouvez outrepasser le fichier de configuration en utilisant quelque chose comme -f-, --no-f, -f=no ou de nombreuses autres variantes.

-t
--tempdir
Répertoire temporaire dans lequel écrire les fichiers modèles debconf et les scripts de configuration extraits Balise de configuration: APT::ExtractTemplates::TempDir.
-h
--help
Affiche un court résumé sur les commandes d'usage.
-v
--version
Affiche la version du programme.
-c
--config-file
Fichier de configuration ; désigne le fichier de configuration à utiliser. Le programme lit le fichier de configuration par défaut puis ce fichier de configuration. Voir apt.conf(5) pour des informations sur la syntaxe.
-o
--option
Applique une Option de Configuration ; cela applique une valeur choisie de configuration à une option. La syntaxe est -o Foo::Bar=bar.
 

VOIR AUSSI

apt.conf(5)  

DIAGNOSTICS

apt-extracttemplates retourne zéro en fonctionnement normal, une valeur entre 1 et 100 en cas d'erreur.
