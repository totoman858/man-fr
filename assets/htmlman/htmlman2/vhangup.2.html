NOM

vhangup - Déconnecter virtuellement un terminal.  

SYNOPSIS

#include <unistd.h>

int vhangup(void);  

DESCRIPTION

vhangup simule une déconnexion du terminal courant. Cet appel système est utilisé pour que les autres utilisateurs aient un écran de connexion ``propre''.  

VALEUR RENVOYÉE

vhangup renvoie zéro s'il réussit, ou -1 s'il échoue auquel cas errno contient le code d'erreur.  

ERREURS

EPERM
L'utilisateur n'est pas Super-User
 

CONFORMITÉ

Cet appel système est spécifique à Linux et ne doit pas être employé dans des programmes destinés à être portables.  

VOIR AUSSI

init(8)
