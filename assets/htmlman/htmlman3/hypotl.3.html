HYPOT

Section: Manuel du programmeur Linux (3)
Updated: 30 juillet 2003
Index Return to Main Contents
 

NOM

hypot, hypotf, hypotl - Distance euclidienne.  

SYNOPSIS

#include <math.h>

double hypot (double x, double y);

float hypot (float x, float y);

long double hypot (long double x, long double y);
 

DESCRIPTION

La fonction hypot() renvoie la valeur sqrt(x*x + y*y). C'est la longueur de l'hypoténuse d'un triangle rectangle de côtés x et y. C'est également la distance entre un point de coordonnées (x, y) et l'origine.  

CONFORMITÉ

SVID 3, BSD 4.3, C99. Les variantes float et long double sont des demandes C99.  

VOIR AUSSI

sqrt(3), cabs(3)  

TRADUCTION

Christophe Blaess, 1996-2003.
