NOM

getpt - Ouvre un pseudo-terminal maître (PTM).  

SYNOPSIS

#include <stdlib.h>

int getpt()
 

DESCRIPTION

getpt() ouvre un pseudo-terminal maître et renvoie son descripteur de fichier. C'est équivalent à open(/dev/ptmx,O_RDWR|O_NOCTTY) sur les systèmes Linux, mais le pseudo-terminal peut être situé ailleurs sur cetains systèmes qui emploient aussi la GlibC Gnu.

 

VALEUR RENVOYÉE

getpt() renvoie un descripteur de fichier ouvert s'il réussit. Sinon il renvoie -1 et remplit errno avec un code d'erreur.  

CONFORMITÉ

getpt() est spécifique à la GlibC GNU.  

VOIR AUSSI

ptmx(4), grantpt(3), unlockpt(3), ptsname(3)
