NOM

groupmod - Modifier un groupe

 

SYNOPSIS

groupmod [-g gid [-o]] [-n nom_du_groupe ] groupe

 

DESCRIPTION

La commande groupmod modifie les fichiers des comptes du système pour refléter les modifications spécifiées sur la ligne de commande. Les options applicables à la commande groupmod sont :
-g gid
La valeur numérique de l'identifiant du groupe. Cette valeur doit être unique, à moins que l'option -o ne soit utilisée. Cette valeur ne doit pas être négative. Les valeurs entre 0 et 499 sont typiquement réservées pour les groupes système. Vous devez modifier manuellement l'identifiant du groupe de fichier pour tous les fichiers dont l'ancien identifiant de groupe est l'identifiant du groupe de fichier.
-n nom_du_groupe
Le nom du groupe sera modifié de groupe en nom_du_groupe.

 

FICHIERS

/etc/group - informations sur les groupes
/etc/gshadow - informations masquées sur les groupes
