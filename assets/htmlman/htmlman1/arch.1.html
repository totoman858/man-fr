NOM

arch - Afficher l'architecture de la machine.  

SYNOPSIS

arch  

DESCRIPTION

arch est équivalent à uname -m

Sur les systèmes Linux actuels, arch affiche "i386", "i486", "i586", "alpha", "sparc", "arm", "m68k", "mips", "ppc".  

VOIR AUSSI

uname (1), uname(2)
