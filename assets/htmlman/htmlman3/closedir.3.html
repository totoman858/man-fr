NOM

closedir - Fermer un repertoire.  

SYNOPSIS

#include <sys/types.h>

#include <dirent.h>

int closedir (DIR *dir);
 

DESCRIPTION

La fonction closedir() ferme le flux répertoire associé à dir. Après cette invocation, le descripteur dir du flux répertoire n'est plus disponible.  

VALEUR RENVOYÉE

La fonction closedir() renvoie 0 si elle réussit, ou -1 si elle échoue, auquel cas errno contient le code d'erreur.  

ERREURS

EBADF
Le descripteur de flux répertoire dir est invalide.
 

CONFORMITÉ

SVID 3, POSIX, BSD 4.3  

VOIR AUSSI

close(2), opendir(3), readdir(3), rewinddir(3), seekdir(3), telldir(3), scandir(3)
