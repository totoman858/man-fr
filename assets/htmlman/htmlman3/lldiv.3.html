NOM

ldiv, lldiv - Quotient et reste d'une division d'entiers longs.  

SYNOPSIS

#include <stdlib.h>

ldiv_t ldiv (long int numer, long int dénom);


lldiv_t lldiv(long long int numer, long long int dénom);
 

DESCRIPTION

La fonction ldiv() divise numer par dénom et renvoie le quotient et le reste dans une structure ldiv_t contenant deux membres entiers longs nommés quot et rem. Le quotient est arrondi vers zéro.  

VALEUR RENVOYÉE

La structure ldiv_t.  

CONFORMITÉ

SVID 3, BSD 4.3, ISO 9899. lldiv() a été ajouté dans ISO C99.  

VOIR AUSSI

div(3)  

TRADUCTION
