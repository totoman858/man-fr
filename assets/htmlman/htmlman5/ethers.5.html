NOM ethers - Base de données adresses Ethernet vers adresses IP.

 

DESCRIPTION /etc/ethers

contient des adresses Ethernet sur 48 bits et leur adresse IP correspondante, une ligne par adresse IP :

adresse-Ethernet adresse-IP

Les deux éléments sont séparés par une ou plusieurs espaces et/ou tabulations. Un « # » en début de ligne est un commentaire pour tout le reste de la ligne. L'adresse-Ethernet est écrite sous la forme x:x:x:x:x:x, où x est une valeur hexadécimale comprise entre 0 et ff représentant un octet de l'adresse, dans l'ordre du réseau (gros-boutiste). L'adresse-IP peut être soit un nom d'hôte résolu par DNS, soit une adresse en notation décimale pointée.  

EXEMPLES 08:00:20:00:61:CA pal
