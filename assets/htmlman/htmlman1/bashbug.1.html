NOM

bashbug - rapporte un bug de bash  

SYNOPSIS

bashbug [adresse]  

DESCRIPTION

bashbug est un script shell destiné à aider l'utilisateur à composer et envoyer un rapport de bug concernant bash, dans un format standard. bashbug invoque l'éditeur spécifié dans la variable d'environnement EDITOR sur une copie temporaire d'un rapport de bug préformaté. L'utilisateur doit remplir correctement les champs et quitter l'éditeur. bashbug envoie alors le rapport complété à bug-bash@gnu.org, ou à adresse. Si le rapport ne peut être envoyé pour une raison quelconque, il est sauvé dans le fichier dead.bashbug dans le répertoire personnel de l'utilisateur.

Le rapport de bug préformaté est divisé en plusieurs sections. La première section fournit des informations concernant la machine, le système d'exploitation la version de bash et l'environnement de compilation. La seconde section doit être remplie avec la description du bug. La troisième section doit expliquer comment reproduire le bug. La quatrième section, optionnelle, est destinée à un éventuel correctif. Vous êtes encouragé à l'utiliser.  

ENVIRONNEMENT

bashbug utilisera les variables d'environnement suivantes, si elles existent :
EDITOR
Spécifie l'éditeur préféré. Si EDITOR n'existe pas, bashbug utilisera emacs par défaut.
HOME
Répertoire dans lequel le fichier doit être sauvé si l'envoie par mail a échoué.
