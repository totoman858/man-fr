NOM

div - Calculer le quotient et le reste d'une division entière.  

SYNOPSIS

#include <stdlib.h>

div_t div (int numer, int dénom);
 

DESCRIPTION

La fonction div() calcule le rapport numer/dénom et renvoie le quotient et le reste dans une structure de type div_t, qui contient le quotient dans son membre quot et le reste dans son membre rem.  

VALEUR RENVOYÉE

La structure div_t.  

CONFORMITÉ

SVID 3, BSD 4.3, ISO 9899  

VOIR AUSSI

ldiv(3)
