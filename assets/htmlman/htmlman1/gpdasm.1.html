NOM

gpdasm - Désassembleur GNU PIC

 

SYNOPSIS

gpdasm [options] fichier

 

AVERTISSEMENT

Les informations de cette page de manuel sont extraites de la documentation complète de gputils et sont limitées à la signification des options. Pour une documentation complète, reportez-vous au fichier « gputils.ps » (en anglais) situé dans le répertoire doc de gputils.

 

DESCRIPTION

gpdasm est un désassembleur pour les microcontrôleurs PIC (TM) de Microchip (TM). gpdasm est une partie de gputils. Consultez la page manuel gputils(1) pour avoir des détails sur les autres utilitaires GNU PIC.

 

OPTIONS

-c, --mnemonics
Décoder les mnémoniques spéciales.
-h, --help
Afficher un message d'usage.
-i, --hex-info
Produire des informations supplémentaires concernant le fichier hexadécimal d'entrée. Ces informations incluent la taille du fichier et le format hexadécimal.
-l, --list-chips
Afficher la liste des processeurs reconnus.
-m, --dump
Afficher le contenu de la mémoire programme.
-p PROC, --processor PROC
Sélectionner le processeur.
-s, --short
Produire en sortie le format court. Ce format peut être réassemblé par gpasm.
-v, --version
Afficher la version.
-y, --extended
Activer le mode étendu 18xx.
