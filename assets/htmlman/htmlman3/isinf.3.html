ISINF

Section: Manuel du programmeur Linux (3)
Updated: 30 juillet 2003
Index Return to Main Contents
 

NOM

isinf, isnan, finite - Tester si un nombre est infini ou non numérique (NaN).  

SYNOPSIS

#include <math.h>

int isinf (double value);

int isnan (double value);

int isfinite (double value);
 

DESCRIPTION

La fonction isinf() renvoie -1 si value représente moins l'infini, 1 si value représente plus l'infini, et 0 sinon.

La fonction isnan() renvoie une valeur non nulle si value est non-numérique (NaN "not-a-number"), et 0 sinon.

La fonction isfinite() renvoie une valeur non nulle si value est fini et numérique, et 0 sinon.  

NOTE

C99 fournit des macros supplémentaires, indépendantes du type comme fpclassify(), isinf() et isnan().  

CONFORMITÉ

BSD 4.3
