NOM

setup - Configurer les périphériques, les systèmes de fichiers et monter le système de fichiers racine.  

SYNOPSIS

#include <unistd.h>

_syscall0(int, setup);

int setup(void);  

DESCRIPTION

setup est appelé une fois dans linux/init/main.c. Il appelle les fonctions d'initialisation des périphériques et des systèmes de fichiers configurés dans le noyau. Ensuite il monte, le système de fichiers racine.

Aucun processus ne peut appeler setup. Même un processus avec des privilèges Super-User déclenchera une erreur EPERM.  

VALEUR RENVOYÉE

setup renvoie toujours -1 pour un processus utilisateur.  

ERREURS

EPERM
Toujours déclenchée pour un processus utilisateur.
 

CONFORMITÉ

Cette fonction est spécifique Linux et ne doit pas être employée dans un programme portable, ni même dans aucun programme du tout. Depuis Linux 2.1.121 cette fonction n'existe plus. La séquence d'appel a changé, elle a d'abord eu un paramètre unique void * BIOS et plus tard un paramètre unique int magic.
