NOM

apt-show-source - Énumère les paquets-sources.  

DESCRIPTION

Ce programme réalise une analyse syntaxique des listes de APT des paquets de source et du fichier d'état de DPKG et liste tous les paquets avec un numéro de version supérieur à celui installé.

Il peut se montrer très utile si les entrées « deb » dans votre fichier sources.list de APT pointent vers la version stable et les entrées « deb-src » pointent vers l'instable. Avec ce programme, vous pouvez aisément savoir s'il y a une nouvelle version de par ex. programme XXXX dans l'instable.  

PARAMÈTRES EN LIGNE DE COMMANDE

Les paramètres en ligne de commandes optionnels sont le fichier d'état de DPKG à utiliser, le chemin d'accès des fichiers listes de APT et le nom d'un paquet. Il y a aussi des options à montrer : tous les paquets de sources, messages longs, version seule et l'aide de la ligne de commande.  

OPTIONS


-stf FICHIER, --status-file=FICHIER

Lit les paquets installés à partir de FICHIER à la place de /var/lib/dpkg/status.

-ld DOSSIER, --list-dir=DOSSIER

DOSSIER spécifie le chemin d'accès aux fichiers listes de APT, /var/state/apt/lists/ par défaut.

-p PAQUET, --package=PAQUET

Affiche les données de version du paquet installé/paquet de source pour PAQUET.

--version-only

Affiche seulement la version si utilisée avec --package.

-a, --all

Affiche tous les paquets sources disponibles avec la version.

-v, --verbose

Affiche de longs messages.

-h, --help

Affiche l'aide en ligne de commande.
