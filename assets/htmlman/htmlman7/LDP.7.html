NOM

LDP - Introduction à la documentation du LDP  

SYNOPSIS

Le Linux Documentation Project (LDP) fournit un large ensemble de documentations libres à la communauté Linux, incluant des guides, des FAQs, des HOWTOs et des pages de manuel.

 

AUTEURS

Les divers documents dans les archives du LDP sont maintenus par des auteurs individuels, indiqués au début de chaque document. Si vous avez des questions ou des informations pour un document, nous vous encourageons à vous mettre directement en rapport avec les auteurs.

 

PAGES WEB

Le LDP a son propre site Web, comme les nombreux projets de traduction qui sont accessibles à partir du site Web principal à
http://www.tldp.org/
que vous pouvez glisser dans vos signets favoris.

 

PAGES DE MANUEL

Une page Web avec l'état des pages de manuel et les traductions se trouve à l'adresse.
http://www.win.tue.nl/~aeb/linux/man/
[NdT] Pour les versions françaises, une adresse plus à jour est http://perso.club-internet.fr/~ccb/.

 

LISTES DE DIFFUSION

Le LDP a plusieurs listes de diffusion, les plus utiles pour les auteurs étant :

<ldp-announce@lists.LinuxDoc.org> Les annonces du projet LDP.
<ldp-discuss@lists.LinuxDoc.org>
Discussions générales sur le projet LDP.

Pour rejoindre ces listes, envoyez un mail avec "subscribe" dans le corps du message à <ldp-announce-request@lists.LinuxDoc.org> ou <ldp-discuss-request@lists.LinuxDoc.org> respectivement. Ces listes sont également archivées sur le site LinuxDoc.

 

FICHIERS

La plupart des distributions incluent les HOWTOs et les mini-HOWTOs lors de l'installation.
/usr/doc/ (ancien emplacement de la documentation)
/usr/share/doc/ (nouvel emplacement de la documentation)
/usr/share/doc/HOWTO/ (fichiers HOWTO)
/usr/share/doc/HOWTO/mini/ (fichiers mini-HOWTO)
 

VOIR AUSSI

man(1), xman(1x), info(1).

Les pages info sont lisibles avec emacs(1) ou info(1).
