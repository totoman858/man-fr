NOM

znew - recompacter des fichiers .Z en des fichiers .gz.  

SYNOPSIS

znew [ -ftv9PK] [ nom.Z ... ]  

DESCRIPTION

znew recompacte des fichiers au format .Z (compress) dans le format .gz (gzip). Si vous voulez recompacter un fichier qui est déjà au format gzip, renommez-le pour imposer une extension .Z et appliquez ensuite znew.  

OPTIONS

-f
Forcer le recompactage du format .Z au format .gz même si un fichier .gz de même nom existe déjà.
-t
Tester les nouveaux fichiers avant de supprimer les originaux.
-v
Verbeux. Afficher le nom et le pourcentage de réduction pour chaque fichier compacté.
-9
Utiliser la méthode de compression la plus lente (compression optimale).
-P
Utiliser des tubes lors de la conversion pour réduire l'utilisation d'espace disque.
-K
Conserver un fichier .Z quand il est plus petit que le fichier .gz
 

VOIR AUSSI

gzip(1), zmore(1), zdiff(1), zgrep(1), zforce(1), gzexe(1), compress(1)  

BOGUES

znew ne préserve pas l'horodate avec l'option -P si cpmod(1) n'est pas disponible, ou si touch(1) ne supporte pas l'option -r.  

TRADUCTION

Frédéric Delanoy, 2001.
