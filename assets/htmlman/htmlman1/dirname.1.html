DIRNAME

Section: Manuel de l utilisateur Linux (1)
Updated: 30 juillet 2003
Index Return to Main Contents
 

NOM

dirname - Ne conserver que la partie répertoire d'un chemin d'accès.  

SYNOPSIS

dirname nom_de_fichier
dirname {--help,--version}  

DESCRIPTION

Cette page de manuel documente la version GNU de dirname.

dirname élimine le dernier composant délimité par un slash du nom_de_fichier.

Si le nom_de_fichier n'a qu'un seul composant, dirname affiche `.' (ce qui signifie `répertoire en cours').  

OPTIONS

--help
Afficher un message d'aide sur la sortie standard et terminer normalement.
--version
Afficher un numéro de version sur la sortie standard et terminer normalement.
