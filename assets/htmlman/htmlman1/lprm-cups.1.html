NOM

lprm - Annule des travaux d'impression  

SYNOPSIS

lprm [ -E ] [ - ] [ -P destination ] [ ID(s) des travaux d'impression ]  

DESCRIPTION

lprm annule des travaux d'impression en attente. L'option -P spécifie l'imprimante ou la classe de destination.

Si aucun argument n'est fourni, c'est le travail en cours sur la destination par défaut qui est annulé. Il est possible de fournir un ou plusieurs ID de travaux d'impression à annuler, ou utiliser l'option - pour annuler tous les travaux.

L'option -E force le cryptage lors de la connexion au serveur.  

COMPATIBILITÉ

La version CUPS de lprm est compatible avec la version standard Berkeley.
