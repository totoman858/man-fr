NOM

sysctl.conf - fichier de configuration pour sysctl(8)  

DESCRIPTION

sysctl.conf est un fichier très simple contenant les valeurs qui doivent être mise en place par sysctl(8). La syntaxe est la suivante : 

# commentaire
; commentaire

  élément  =   valeur

Notez que les lignes vides sont ignorées et ainsi que les espaces avant et après un élément ou une valeur. Une espace dans une valeur fait bien partie de la valeur. Les lignes commençant par un # ou un ; sont des commentaires et sont ignorées.  

EXEMPLE

# exemple de fichier sysctl.conf
#

  kernel.domainname = exemple.com
; celle-ci a une espace qui sera écrite par la commande sysctl !
  kernel.modprobe = /sbin/mod probe
