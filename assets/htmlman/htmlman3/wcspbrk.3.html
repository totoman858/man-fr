NOM

wcspbrk - Rechercher un ensemble de caractères larges dans une chaîne.  

SYNOPSIS

#include <wchar.h>

wchar_t *wcspbrk (const wchar_t *wcs, const wchar_t *accept);
 

DESCRIPTION

La fonction wcspbrk est l'équivalent pour les caractères larges de la fonction strpbrk. Elle recherche la première occurence d'un caractère large se trouvant dans accept au sein de la chaîne wcs.  

VALEUR RENVOYÉE

La fonction wcspbrk renvoie un pointeur sur la première occurence au sein de wcs d'un caractère large contenu dans accept. Si wcs n'en contient aucun, elle renvoie NULL.  

CONFORMITÉ

ISO/ANSI C, UNIX98  

VOIR AUSSI

strpbrk(3), wcscspn(3), wcschr(3)
