NOM

ram - Périphérique disque virtuel.  

DESCRIPTION

Le périphérique ram est un fichier bloc utilisé pour accéder de manière directe au disque virtuel.

Il est typiquement créé ainsi :

mknod -m 660 /dev/ram b 1 1
chown root:disk /dev/ram

 

FICHIERS

/dev/ram  

VOIR AUSSI

mknod(1), chown(1), mount(8)
