NOM

wcsrchr - Rechercher un caractère large dans une chaîne.  

SYNOPSIS

#include <wchar.h>

wchar_t *wcsrchr (const wchar_t *wcs, wchar_t wc);
 

DESCRIPTION

La fonction wcsrchr est l'équivalent pour les caractères larges de la fonction strrchr. Elle recherche la dernière occurence de wc dans la chaîne de caractères larges pointée par wcs.  

VALEUR RENVOYÉE

La fonction wcsrchr renvoie un pointeur sur la dernière occurence de wc dans la chaîne de caractères larges wcs, ou NULL si wc ne s'y trouve pas.  

CONFORMITÉ

ISO/ANSI C, UNIX98  

VOIR AUSSI

strrchr(3), wcschr(3)
