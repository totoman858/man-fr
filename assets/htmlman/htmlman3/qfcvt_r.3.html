NOM

ecvt_r, fcvt_r, qecvt_r, qfcvt_r - Convertir un réel en chaîne.  

SYNOPSIS

#include <stdlib.h>

char *ecvt_r(double nombre, int nb_chiffres, int *decpt, int *sign, char *tampon, size_t longueur);

char *fcvt_r(double nombre, int nb_chiffres, int *decpt, int *sign, char *tampon, size_t longueur);

char *qecvt_r(long double nombre, int nb_chiffres, int *decpt, int *sign, char *tampon, size_t longueur);

char *qfcvt_r(long double nombre:, int nb_chiffres, int *decpt, int *sign, char *tampon, size_t longueur);  

DESCRIPTION

Les fonctions ecvt_r, fcvt_r, qecvt_r et qfcvt_r sont identiques à ecvt, fcvt, qecvt et qfcvt, respectivement, sauf qu'elles ne renvoient pas leurs résultats dans un tampon statique, mais utilisent à la place le tampon inidiqué dont la longueur est précisée. Voir ecvt(3) et qecvt(3).  

NOTES

Ces fonctions sont obsolètes. L'utilisation de sprintf() est recommandée à leur place.  

CONFORMITÉ

Ce sont des extensions GNU.  

VOIR AUSSI

ecvt(3), qecvt(3), sprintf(3)
