NOM

bindresvport - Affecter une socket à un port privilégié.  

SYNOPSIS

#include <sys/types.h>
#include <netinet/in.h>

int bindresvport(sd, sin)
int sd;
struct sockaddr_in *sin;
 

DESCRIPTION

bindresvport() sert à affecter un descripteur de socket à un port IP privilégié, c'est à dire dans l'intervalle 0-1023. La routine renvoie 0 si elle réussit, ou -1 en cas d'echec, avec errno contenant le code d'erreur.

Seul le Super-utilisateur peut affecter une socket à un port privilégié. Cette fonction échouera pour tous les autres utilisateurs.
