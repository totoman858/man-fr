NOM

tty - Afficher le nom du terminal associe à l'entrée standard.  

SYNOPSIS

tty [-s] [--silent] [--quiet] [--help] [--version]  

DESCRIPTION

Cette page de manuel documente la version GNU de tty.

tty affiche le nom de fichier du tty associé à son entrée standard. Il affiche `not a tty' si l'entrée standard n'est pas un terminal.

Son code de retour vaut : 

0 si l'entrée standard est un tty
1 si l'entrée standard n'est pas un tty
2 si un argument est incorrect
3 s'il y a eu une erreur d'écriture
 

OPTIONS

--help
Afficher un message d'aide sur la sortie standard et se terminer normalement.
-s, --silent, --quiet
Ne rien afficher, renvoyer simplement le code de retour.
--version
Afficher un numéro de version sur la sortie standard et se terminer normalement.
