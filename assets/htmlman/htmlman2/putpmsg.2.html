NOM

afs_syscall, break, ftime, getpmsg, gtty, lock, mpx, prof, profil, putmsg, security, stty, ulimit - Appels systèmes non implémentés.  

DESCRIPTION

Ces appels systèmes ne sont pas implémentés dans le noyau Linux 2.4.  

VALEUR RENVOYÉE

Tous ces appels systèmes renvoient -1 et errno contient le code d'erreur ENOSYS.  

NOTE

Remarquez que ftime(3), profil(3), et ulimit(3) sont implémentés comme des fonctions de bibliothèque.

Certains appels-systèmes comme alloc_hugepages(2), free_hugepages(2), ioperm(2), iopl(2), et vm86(2) n'existent que sur certaines architectures.

Certains appels-systèmes comme ipc(2), et {create,init,delete}_module(2) n'existent que si le noyau Linux a été compilé avec les options de support adéquates.  

VOIR AUSSI

obsolete(2)
