NOM

gplink - Éditeur de liens GNU PIC

 

SYNOPSIS

gplink [options] [objets] [bibliothèques]

 

AVERTISSEMENT

Les informations de cette page de manuel sont extraites de la documentation complète de gputils et sont limitées à la signification des options. Pour une documentation complète, reportez-vous au fichier « gputils.ps » (en anglais) situé dans le répertoire doc de gputils.

 

DESCRIPTION

gplink conbine un nombre de fichiers objets et archives, reloge leurs données, et lie leurs références symboles. Il produit en sortie un exécutable pour les microcontrôleurs PIC (TM) de Microchip (TM). gplink est une partie de gputils. Consultez la page manuel gputils(1) pour avoir des détails sur les autres utilitaires GNU PIC.

 

OPTIONS

-a FMT, --hex-format FMT
gplink reconnaît les formats de fichiers hexadécimaux inhx8m, inhx8s, inhx16, et inhx32. Cette option contrôle quel format de fichier hexadécimal est utilisé. La valeur par défaut est inhx32.
-c, --object
Produire en sortie un objet exécutable.
-d, --debug
Produire des messages de débogage.
-f VALEUR, --fill VALEUR
Remplir la mémoire programme inutilisée et non protégée avec VALEUR. VALEUR supplante la valeur de remplissage du script éditeur de liens. VALEUR ne doit pas être supérieure à 0xffff. VALEUR est en hexadécimal et doit être précédée par « 0x ».
-h, --help
Afficher un message d'usage.
-I RÉP, --include RÉP
Spécifier le répertoire d'inclusion d'en-tête.
-m, --map
Produire en sortie un fichier d'équivalence.
-o FICHIER, --output FICHIER
Nom alternatif du fichier de sortie.
-q, --quiet
Supprimer tout ce qui est envoyé sur la sortie standard.
-r, --use-shared
Essayer de reloger des sections de données non partagées vers la mémoire partagée si la translation échoue.
-s FICHIER, --script FICHIER
Choisir le script éditeur de lien.
-v, --version
Afficher la version.
