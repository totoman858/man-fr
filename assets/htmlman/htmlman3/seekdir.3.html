NOM

seekdir - Positionner le pointeur de flux répertoire pour le prochain appel readdir().  

SYNOPSIS

#include <dirent.h>

void seekdir (DIR *dir, off_t offset);
 

DESCRIPTION

La fonction seekdir() fixe la position du pointeur de flux répertoire d'où le prochain appel readdir() demarrera. seekdir() doit être employée avec une position offset renvoyée précédemment par telldir().  

VALEUR RENVOYÉE

La fonction seekdir() ne renvoie pas de valeur.  

CONFORMITÉ

BSD 4.3  

VOIR AUSSI

lseek(2), opendir(3), readdir(3), closedir(3), rewinddir(3), telldir(3), scandir(3).
