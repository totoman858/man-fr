NOM

wcwidth - Déterminer le nombre de colonnes nécessaire pour un caractère large.  

SYNOPSIS

#include <wchar.h>

int wcwidth (wint_t c);
 

DESCRIPTION

La fonction wcwidth renvoie le nombre de colonnes nécessaires pour représenter le caractère large c. Si c est un caractère large imprimable, la valeur est au moins 0. Si c est L'\0', la valeur renvoyée est 0. Sinon -1 est renvoyé.  

VALEUR RENVOYÉE

La fonction wcwidth renvoie le nombre de colonnes pour c.  

CONFORMITÉ

UNIX98, POSIX.1-2001. La GlibC avant 2.2.5 utilisait le prototype 
int wcwidth(wint_t c);
 

VOIR AUSSI

wcswidth(3), iswprint(3)  

NOTES

Le comportement de wcwidth dépend de la valeur courante de la variable d'environnement LC_CTYPE correspondant à la catégorie de locales correspondate.
