NOM

acct - Fichier de comptabilité d'exécution.  

SYNOPSIS

#include <sys/acct.h>  

DESCRIPTION

Le noyau maintient une structure d'information de comptabilité pour tous les processus. Si un processus se termine, et si la comptabilité est activée, le noyau appelle la fonction acct(2) pour préparer puis ajouter l'enregistrement du processus dans le fichier de comptabilité. La structure struct acct est décrite dans /usr/include/linux/acct.h.  

VOIR AUSSI

acct(2), sa(1).
