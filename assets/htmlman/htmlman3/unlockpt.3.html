UNLOCKPT

NOM

unlockpt - Déverouiller une paire de pseudo-terminaux maître/esclave.  

SYNOPSIS

#include <stdlib.h>

int unlockpt(int fd);
 

DESCRIPTION

La fonction unlockpt() déverouille le périphérique pseudo-terminal (pty) esclave correspondant au pty maître indiqué par fd.

unlockpt() devrait être invoquée avant d'ouvrir le côté esclave du pty.  

VALEUR RENVOYÉE

Lorsqu'elle réussit, la fonction unlockpt() renvoie 0. Sinon, elle renvoie -1 et remplit errno avec le code d'erreur.  

ERREURS

EBADF
L'argument fd n'est pas un descripteur ouvert en écriture.
EINVAL
L'argument fd n'est pas associé avec un pseudo-terminal maître.
 

CONFORMITÉ

POSIX 1003.1-2001.  

VOIR AUSSI

grantpt(3), ptsname(3)
