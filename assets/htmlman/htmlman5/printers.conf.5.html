NOM

printers.conf - Fichier de configuration des imprimantes CUPS  

DESCRIPTION

Le fichier printers.conf définit les imprimantes locales de la machine. Il est habituellement situé dans le répertoire /etc/cups et est généré automatiquement par le programme cupsd(8) lors de l'ajout ou de la suppression d'imprimantes.

Chaque ligne du fichier est une directive de configuration, une ligne vide ou un commentaire (ligne commençant par la caractère #).  

DIRECTIVES

Accepting

Définit si l'imprimante accepte ou non les travaux d'impression.
Info

Définit un commentaire de description de l'imprimante.
Location

Définit un commentaire concernant la localisation physique de l'imprimante.
DeviceURI

Définit l'URI de l'imprimante.
<Printer name> ... </Printer>

Définit une imprimante.
State

Définit l'état initial de l'imprimante (Idle ou Stopped).
StateMessage

Définit le message associé à cet état.
