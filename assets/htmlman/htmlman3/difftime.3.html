NOM

difftime - Calculer des intervalles de temps.  

SYNOPSIS

#include <time.h>

double difftime (time_t time1, time_t time0);
 

DESCRIPTION

La fonction difftime() renvoie le nombre de secondes écoulées entre les instants time0 et time1. Les deux arguments sont de type time_t, ce qui représente le temps écoulé depuis le 1er Janvier 1970 à 00h 00m 00s, en Temps Universel TU.  

CONFORMITÉ

SVID 3, BSD 4.3, ISO 9899  

NOTES

Cette fonction est réclamée par le C ANSI. Sur un système POSIX, time_t est un type arithmétique, et on peut définir directement 
#define difftime(t1,t0) (double)(t1 - t0)
si les risques de débordement dans la soustraction ne sont pas gènants.  

VOIR AUSSI

date(1), gettimeofday(2), time(2), ctime(3), gmtime(3), localtime(3)
