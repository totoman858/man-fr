NOM

ctrlaltdel - Définir la fonction de la combinaison Ctrl-Alt-Del  

SYNOPSIS

ctrlaltdel hard | soft  

DESCRIPTION

Basé sur l'examen du code de linux/kernel/sys.c, il est clair qu'il existe deux fonctions prises en charge par la combinaison des touches Ctrl-Alt-Del : un redémarrage matériel qui redémarre immédiatement l'ordinateur sans appeler sync(2) et sans aucune autre préparation ; et un redémarrage logiciel qui envoye le signal SIGINT (interruption) au processus init (processus dont le PID est toujours 1). Si cette option est choisie, le programme init(8) doit prendre en charge cette fonctionnalité. Puisqu'il y a maintenant plusieurs programmes init(8) dans la communauté Linux, veuillez consultez la documentation de la version que vous utilisez.

ctrlaltdel est généralement utilisé dans le fichier /etc/rc.local.  

FICHIERS

/etc/rc.local  

VOIR AUSSI
