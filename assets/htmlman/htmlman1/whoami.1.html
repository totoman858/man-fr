NOM

whoami - Afficher l'U-ID effectif.  

SYNOPSIS

whoami [--help] [--version]  

DESCRIPTION

Cette page de manuel documente la version GNU de whoami.

whoami affiche le nom d'utilisateur associé à l'U-ID effectif en cours. C'est équivalent à la commande `id -un'.  

OPTIONS

--help
Afficher un message d'aide sur la sortie standard et se terminer normalement.
--version
Afficher un numéro de version sur la sortie standard et se terminer normalement.
