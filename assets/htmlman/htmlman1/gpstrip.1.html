NOM

gpstrip - retire les symboles d'un fichier objet

 

SYNOPSIS

gpstrip [options] [fichier]

 

AVERTISSEMENT

Les informations de cette page de manuel sont extraites de la documentation complète de gputils et sont limitées à la signification des options. Pour une documentation complète, reportez-vous au fichier « gputils.ps » (en anglais) situé dans le répertoire doc de gputils.

 

DESCRIPTION

gpstrip retire les symboles des fichiers objet GNU PIC gpstrip est une partie de gputils. Consultez la page manuel gputils(1) pour avoir des détails sur les autres utilitaires GNU PIC.

 

OPTIONS

-g, --strip-debug
Éliminer les symboles de débogage.
-h, --help
Afficher ce message d'usage.
-k SYMBOL, --keep-symbol SYMBOLE
Conserver le symbole SYMBOLE.
-n SYMBOL, --strip-symbol SYMBOLE
Supprimer le symbole SYMBOLE.
-o FICHIER,, --output FICHIER,
Nom alternatif du fichier de sortie.
-p, --preserve-dates
Préserver les dates.
-r SECTION, --remove-section SECTION
Retirer la section SECTION.
-s, --strip-all
Retirer tous les symboles.
-u, --strip-unneeded
Retirer les symboles qui ne sont pas nécessaires à la translation (NdT : relocation).
-v, --version
Afficher la version.
-x, --discard-all
Retirer les symboles non globaux.
