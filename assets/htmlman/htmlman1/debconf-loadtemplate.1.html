NOM

debconf-loadtemplate - charge un questionnaire dans la base de données de debconf  

SYNOPSIS

 debconf-loadtemplate propriétaire fichier [fichier ..]
 

DESCRIPTION

Ce programme charge un ou plusieurs questionnaires dans la base de données de debconf. Le premier paramètre spécifie le propriétaire du questionnaire (généralement, le propriétaire est le nom du paquet). Les paramètres restants sont les questionnaires à charger.  

ATTENTION

Ce programme ne devrait jamais être utilisé par les scripts de maintenance d'un paquet qui utilise debconf ! Il devrait, cependant, être utile pour le debogage ou pour remplir la base de données de debconf.  

VOIR AUSSI
