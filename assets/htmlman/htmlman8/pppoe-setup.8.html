NOM

pppoe-setup - Script shell pour configurer le client PPPoE de Roaring Penguin.  

SYNOPSIS

pppoe-setup

 

DESCRIPTION

pppoe-setup est un script shell qui vous invite à fournir quelques informations afin d'établir un script de configuration /etc/ppp/pppoe.conf pour les scripts pppoe-start, pppoe-stop et pppoe-connect.

 

AUTEUR
