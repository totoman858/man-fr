NOM

netreport - demande à être prévenu d'un changement concernant une interface réseau  

SYNOPSIS

netreport [-r]  

DESCRIPTION

netreport demande au scripts de gestion du réseau d'envoyer le signal SIGIO au processus qui a lancé netreport dès lors que l'état d'une interface réseau change.  

OPTIONS

-r
Supprime la requête courante (s'il en existe une) concernant le processus appelant.

 

NOTES

Si un programme n'appelle pas netreport avec l'option -r avant qu'il ne se termine, et qu'un autre processus est créé avec le même PID avant qu'un changement concernant l'état d'une interface réseau change, il est possible que ce nouveau processus recoive un SIGIO non demandé.
