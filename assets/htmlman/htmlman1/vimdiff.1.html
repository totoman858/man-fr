NOM

vimdiff - ouvre dans Vim deux ou trois versions d'un fichier et affiche les différences  

SYNOPSIS


vimdiff [options] fichier1 fichier2 [fichier3]

gvimdiff  

DESCRIPTION

Vimdiff ouvre dans Vim deux (ou trois) fichiers. Chaque fichier est ouvert dans sa propre fenêtre. Les différences entre les fichiers sont mises en surbrillance. C'est pratique pour visualiser les changements et reporter les modifications d'une version à l'autre d'un fichier.

Voir vim(1) pour les informations propres à Vim.

Si l'interface graphique est disponible, elle sera utilisée en tapant gvimdiff

Dans chaque fenêtre, l'option « diff » est utilisée afin de mettre les différences en surbrillance.
Pour donner un bon aspect au texte, les options « wrap » et « scrollbind » sont utilisées.
L'option « foldmethod » vaut « diff », ceci regroupe les lignes non modifiées dans un pli. Afin de repérer, d'ouvrir et de fermer facilement les plis, l'option « foldcolumn » vaut deux.  

OPTIONS

La séparation verticale est utilisée pour aligner les lignes, comme si l'option « -O » était spécifié. Si vous désirez plutôt utiliser une séparation horizontale, spécifiez l'option « -o ».

Pour toute autre option, voir vim(1).
