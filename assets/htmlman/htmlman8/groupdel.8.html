NOM

groupdel - Supprime un groupe

 

SYNOPSIS

groupdel groupe

 

DESCRIPTION

La commande groupdel modifie les fichiers des comptes système, effaçant toutes les entrées qui font référence à groupe. Le groupe cité doit exister.

Vous devez vérifier manuellement tous les systèmes de fichiers pour être certain qu'il ne reste plus un seul fichier appartenant au groupe cité.

 

MISE EN GARDE

Vous ne devez pas supprimer le groupe primaire d'un utilisateur existant. Vous devez d'abord supprimer l'utilisateur avant de supprimer le groupe.

 

FICHIERS

/etc/group - informations sur les groupes
/etc/gshadow - informations masquées sur les groupes
