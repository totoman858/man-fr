NOM

zforce - imposer une extension « .gz » à tous les fichiers gzippés.

 

SYNOPSIS

zforce [ nom ... ]

 

DESCRIPTION

zforce impose une extension .gz à tous les fichiers gzippés afin que gzip ne les compacte pas deux fois. Cela peut être utile pour des fichiers dont le nom a été tronqué après un transfert de fichier. Sur les systèmes où la longueur maximale des noms de fichiers est fixée à 14, le nom original est tronqué pour faire de la place pour le suffixe « .gz ». Par exemple, 12345678901234 est renommé en 12345678901.gz. Un nom de fichier comme foo.tgz est laissé inchangé.

 

VOIR AUSSI

gzip(1), znew(1), zmore(1), zgrep(1), zdiff(1), gzexe(1)
